CSS / 选择器

发表于 2021-01-30 更新于 2021-02-19 分类于 2102：CSS 阅读次数：
本文字数： 1.9k

要使用 CSS 对 HTML 页面元素实现一对一，一对多，多对一的控制，就要用到选择器。

一、基本选择器#

1、元素选择器#

也叫标签名选择器，通过元素标签名字选择页面中所有这个标签的元素。

2、class 选择器#

class 属性用来给元素分组，是允许多个元素拥有的，且一个元素可以拥有多个class。
通过 . + 元素的class属性进行选择。

3、id 选择器#

id相对class来说，是不允许重复的，且一个元素只能有一个id属性。

id 属性命名规范：

  • 以字母开头
  • 不出现汉字、空格
  • 不推荐带有ad 相关的英文单词，容易被浏览器广告过滤插件屏蔽
  • 见名知意

通过 # + 元素的id属性进行选择。

4、通配符选择器#

表示选择所有的元素。
几乎不用，影响性能。

二、关系选择器#

1、群组选择器（并集选择器）#

用 ,就能对多个选择器一起设置样式。

2、交集选择器#

由两个选择器直接连接构成，其结果是选中二者各自元素范围的交集。其中第一必须是标签选择器，第二个必须是类选择器或者id选择器，两个选择器之间不能有空格，必须连续书写。
如：p#xxx 或者 p.xxx。

3、子级选择器#

用>隔开。如 A>B，在A元素中寻找子元素是B的。

4、后代选择器#

用空格隔开。如 A B，意思是在A元素中寻找后代是B的元素。

5、相邻兄弟选择器#

相邻兄弟选择器 A+B，意思是在A元素下紧贴这A的并且是B才能被选中。

6、通用兄弟选择器#

通用兄弟选择器 A~B，在A元素下边的兄弟元素，并且是B就能被选中。

三、属性选择器#

选择器 描述
E[attr] 有属性attr的所有E元素
E[attr='val'] 属性attr的值是val的元素
E[attr^='val'] 属性attr的值以val开头的元素
E[attr$='val'] 属性attr的值以val结尾的元素
E[attr*='val'] 属性attr的值包含val字符串的元素

四、伪类选择器#

1、动态伪类选择器#

  • 锚伪类
    link 未访问
    visited 已访问
    hover 鼠标移动到
    active 选定
  • 用户行为伪类
    focus 获取焦点
  • 目标伪类：target（IE9及以上）

2. 状态伪类选择器#

选择器 描述
:enabled 匹配表单中可用的元素
:disabled 匹配表单中禁用的元素
:checked 匹配表单中被选中的radio（单选框）或checkbox（复选框）元素
:invalid 无效的元素

3. 结构性伪类选择器#

选择器 描述
:first-child 选择第一个子元素
:last-child 选择最后一个子元素
x:first-of-type 选择其子元素中第一个x元素
x:last-of-type 选择其子元素中最后一个x元素
:nth-child(n) 选择一个或多个特定的子元素 （第一个编号为1）
:nth-last-child(n) 选择一个或多个特定的子元素，从最后一个子元素开始算
:only-child 匹配父元素下仅有的一个子元素
:empty 选择没有子元素的
:not(x) 选择 x 以外的所有元素

参考

  • Pseudo-classes

五、伪元素选择器#

:before 在被选元素的内容前面插入内容。
:after 在被选元素的内容后面插入内容。

:first-letter 向文本的第一个字母添加特殊样式。
:first-line 向文本的首行添加特殊样式。

用 content 属性插入内容。

六、选择器的优先级与权重#

1. !important#

2. 优先级顺序#

从上到下，依次增加。

  • 默认 / 继承（inherit）的样式
  • 通配符
  • 标签选择器
  • 类选择器
  • id 选择器
  • 行内书写的样式
  • !important

当优先级相同时，后写的样式优先级更高。
