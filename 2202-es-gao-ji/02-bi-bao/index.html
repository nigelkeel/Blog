闭包#

1、什么是闭包#

闭包是一个包含被引用局部变量的 closure 对象，存在嵌套的内部函数中。

2、怎么产生闭包 & 生命周期#

必须满足三个条件

  • 函数嵌套函数
  • 内部函数调用外部函数的局部变量（引入全局变量是不会产生闭包的）
  • 调用外部函数

生命周期

  • 内部函数定义执行完成就产生
  • 内部函数成为垃圾对象就销毁

3、闭包的作用与缺点#

作用

  • 延长了局部变量的存活时间
  • 让函数外部操作函数内部的数据

缺点

  • 用太多可能会导致内存泄露（因为占用内部空间不释放）

4、闭包的应用#

  • 迭代器
  • React 组件高阶函数（为了复用函数，给内部函数传参）
  • Vue 原理中存在（双向数据绑定原理）
