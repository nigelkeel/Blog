310203：json-server

发表于 2021-01-31 分类于 3102：接口

json-server#

用来快速搭建模拟的 REST API 的工具包。

1、开始#

（1）安装

npm i json-server -g

（2）查看版本

json-server -v

（3）准备一个 db.json

// db.json
{
    "sword": [{
        "id": 0,
        "name": "木剑",
        "attr": {
            "atk": 1,
            "parry": 1
        }
    },{
        "id": 1,
        "name": "铜剑",
        "attr": {
            "atk": 5,
            "parry": 3
        }
    },{
        "id": 2,
        "name": "铁剑",
        "attr": {
            "atk": 8,
            "parry": 6
        }
    }{
        "id": 3,
        "name": "钢剑",
        "attr": {
            "atk": 12,
            "parry": 10
        }
    }],
    "shield": [{
        "id": 0,
        "name": "木盾",
        "def": 5
    }]
}

（4）配置服务器

// package.json
// > npm run server
{
  "scripts": {
    "s": "json-server -w db.json -H 127.0.0.1 -p 8888"
  },
  "name": "server",
  "version": "1.0.0",
  "dependencies": {}
}
// json-server.json
{
  "port": 80,
  "watch": true,
  "host": "127.0.0.1",
  // 设置静态目录
  "static": "./public",
  "read-only": false,
  // 允许跨域
  "no-cors": false,
  "no-gzip": false,
  "delay": 0
}

（5）开启服务器

json-server --watch db.json
json-server --watch db.json --host 127.0.0.1 --port 8888
npm run server
