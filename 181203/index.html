VSCode 插件#

一、插件一览#

  1. Chinese (Simplified) Language Pack for Visual Studio Code - 中文语言包
  2. Markdown All in One
  3. Markdown Perview Eehanced
  4. HTML CSS Support：智能提示CSS类名以及id
  5. Python
  6. Python for VSCode
  7. Beautify 代码格式化 --> Prettier
  8. Prettier
  9. Code Spell Checker（推荐） - 代码拼写检查
  10. TabNine
  11. Better Comments - 更好的注释
  12. Rainbow Brackets
  13. Auto Rename Tag：自动完成另一侧标签的同步修改
  14. Easy LESS - 自动编译LESS
  15. Live Server - HTML 服务器
  16. Path Autocomplete
  17. Path Intellisense：自动提示文件路径，支持各种快速引入文件
  18. Draw.io Integration：画图
  19. A-super-comprehensive：代码补全合集
  20. Reactjs code snippets
  21. React Redux ES6 Snippets
  22. ES7 React/Redux/React-Native/JS snippets
  23. Auto Close Tag：自动闭合HTML/XML标签
  24. Bracket Pair Colorizer：给括号加上不同的颜色，便于区分不同的区块，使用者可以定义不同括号类型和不同颜色
  25. Debugger for Chrome：映射vscode上的断点到chrome上，方便调试
  26. GitLens：方便查看git日志，git重度使用者必备
  27. HTML Snippets：智能提示HTML标签，以及标签含义
  28. JavaScript(ES6) code snippets：ES6语法智能提示，以及快速输入，不仅仅支持.js，还支持.ts，.jsx，.tsx，.html，.vue，省去了配置其支持各种包含js代码文件的时间
  29. open in browser：vscode不像IDE一样能够直接在浏览器中打开html，而该插件支持快捷键与鼠标右键快速在浏览器中打开html文件，支持自定义打开指定的浏览器，包括：Firefox，Chrome，Opera，IE以及Safari
  30. React/Redux/react-router Snippets：React/Redux/react-router语法智能提示
  31. Vetur：Vue多功能集成插件，包括：语法高亮，智能提示，emmet，错误提示，格式化，自动补全，debugger。vscode官方钦定Vue插件，Vue开发者必备。
  32. vscode-icon：让 vscode 资源树目录加图标
  33. CSS Peek：使用此插件，你可以追踪至样式表中 CSS 类和 ids 定义的地方。当你在 HTML 文件中右键单击选择器时，选择“ Go to Definition 和 Peek definition ”选项，它便会给你发送样式设置的 CSS 代码。
  34. Color Info：提供你在 CSS中使用颜色的相关信息。你只需在颜色上悬停光标，就可以预览色块中色彩模型的（HEX、 RGB、HSL 和 CMYK）相关信息了。
  35. Window Colors：每个VSCode窗口都可以独特地自动着色。
  36. live server：开启本地服务器
