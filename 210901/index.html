ECharts#

百度出品

文档：https://echarts.apache.org/examples

一、开始#

npm install echarts --save

二、使用步骤#

  1. 引入
  2. 创建一个具有宽高的容器
  3. 创建一个实例，绑定容器节点，并调用 echarts.init 方法
  4. 创建一个配置对象 option ，修改配置
  5. 通过这个实例 .setOption(option) 方法，传入配置对象

三、图表类型#

  • bar 柱状图
  • line 折线图
  • pie 饼图
  • gauge 仪表盘
  • scatter 散点图
