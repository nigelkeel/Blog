一、网页#

1、概念#

  • WWW，World Wide Web，万维网
  • B/S结构，浏览器-服务器结构
  • C/S结构，客户端-服务器结构
  • HTTP协议，超文本传输协议
  • Domain Name，域名，DNS；Domain Name System，域名解析系统
  • W3C，万维网联盟
  • HTML，Hypertext Markup Language，超文本标记语言。
  • CSS，层叠样式表。

2、网页的组成#

网页由三部分组成：

  • 结构 html
  • 表现 CSS
  • 行为 JavaScript

二、浏览器#

1、常见浏览器及其内核#

浏览器 内核 浏览器前缀
IE Tredent -ms-
Safari Webkit -webkit-
Firefox Gecko -moz-
Chrome Blink -webkit-
Edge、Opera Blink -webkit-

注1 Edge之前采用 EdgeHTML 内核，Opera之前使用 Presto 内核。Blink内核是Webkit内核的分支。
注2 Android和iOS移动设备只能用自带系统浏览器内核。
注3 PC端浏览器市场份额Chrome浏览器排名第一，IE第二，二者约占70%。
