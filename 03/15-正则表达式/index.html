JS 正则表达式

发表于 2020-08-21 更新于 2020-09-04 分类于 技术栈

正则表达式（regexp） 是搜索和替换字符串一种强大的方法。在 JS 中，正则表达式通过内置的 RegExp 类的对象来实现，并和字符串集成。

一、基本语法#

正则表达式包含 模式 和可选的 修饰符。

1、创建#

创建一个正则表达式有两种语法：

// 允许从字符串动态构造模式
reg = new RegExp("pattern", "flags");

// 常用
reg2 = /pattern/;
reg3 = /pattern/gmi;

如果要在字符串中搜索，可以使用 search() 。这个方法返回匹配项的索引位置，如果没找到返回-1。

// 例子

let str = "I love you!";
let reg = /love/;

console.log( str.search(reg) ); // 2

2、修饰符#

正则表达式的修饰符影响搜索结果，在 JS 中有五个修饰符：

修饰符 描述
i 搜索不区分大小写
g 搜索会查找所有的匹配项，而不是第一个
m 多行模式
u 开启完整的unicode支持
y 粘滞模式

搜索默认是区分大小写的。

let str = "I love you!";

console.log( str.search(/LOVE/) ); // -1
console.log( str.search(/LOVE/i) ); // 2

3、字符类#

类 描述
. 匹配所有字符（除换行符外）
\d 任何一个数字（digit）
\D 非数字
\s 空格（制表符，换行符）
\S 非空格
\w 拉丁字母或者数字或者下划线。
\W 非单字

4、锚点#

插入符 ^ 匹配文本开头，而美元符 $ 则匹配文本末尾。它们被称为锚点。

5、Unicode#

JavaScript 使用 Unicode 编码（Unicode encoding）对字符串进行编码。大多数字符使用 2 个字节编码，但这种方式只能编码最多 65536 个字符。

这个范围不足以对所有可能的字符进行编码，这就是为什么一些罕见的字符使用 4 个字节进行编码，比如 𝒳 （数学符号 X）或者 😄 （笑脸），一些象形文字等等。

Unicode：修饰符 “u” 和 class \p{…}
