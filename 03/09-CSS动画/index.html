  • 扭曲 skew
  • 改变变换基点 transform-origin

动画分为 过渡动画 和 关键帧动画。

二、过渡动画 transition#

允许CSS的属性在一定的时间区间内平滑过渡。在鼠标点击，鼠标滑过或对元素任何改变中触发，并平滑地以动画改变CSS的属性值。

transition

  • property 参与过渡的属性
  • duration 持续时间
  • timing-function 动画类型 linear 线性匀速
  • delay 延迟时间
transition: height 3s linear 2s

三、关键帧动画 animation#

关键帧动画是控制关键位置，而不是所有位置。

1、@keyframes 定义新关键帧#

@keyframes <name>{
	from/0%
	{
		// ...
	} 
	to/100%
	{
		// ...
	}
}

2、animation#

  • name 动画属性名
  • duration 关键帧持续时间
  • timing-function 动画执行方式（针对于每一个关键帧之间的动画）
  • delay 动画延迟时间
  • iteration-count 动画循环次数，次数或者 infinite（无穷次）
  • direction 控制动画循环反向播放，默认正常播放，alternate，轮流反向播放
