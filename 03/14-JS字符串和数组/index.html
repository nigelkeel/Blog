2、包装对象 +#

在JS中，提供了三个特殊的构造函数Number、String、Boolean，可以使用new来操作，会生成对应类型的对象。（实际开发中不用）

var str = new String("abc");
// 这确实是一个对象
// 被当成对象
// __proto__指向原型对象。原型方法有大量的方法

执行过程：

  1. 声明了一个字符串放入了str中。
  2. 当发现你把str当做对象调用时，js内部会自动调用new String('abc')创建了一个对象（字符串对象，包装对象）；
  3. 然后使用包装对象的属性、方法。当使用完成之后，又会将包装对象转换成字符串。

number、boolean、string 都会经过上面的过程。

普通对象和包装对象的区别：主要针对对象的生命周期。

  • 使用 new 操作符创建的引用类型的实例，在执行流离开之前的作用域中将会一直存在。
  • 自动创建的基本包装类型对象，只存在于那一行代码的执行的一瞬间，然后立即销毁了。

二、字符串方法#

1、字符串查找#

indexOf、lastIndexOf#

indexOf() 正序查找 lastIndexOf() 逆序查找 根据参数字符串，返回指定子字符串的下标位置，如果没找到，返回-1

  • 第1个参数 查找的目标
  • 第2个参数 可选 开始查找的位置

search#

search() 功能和indexOf()相同，定义匹配模式，只有一个参数，没有反向检索和全局模式。

macth#

macth() 找出所有匹配的子字符串，并以数组的形式返回，如果匹配模式没有指定全局，则进行单次匹配，如果没找到返回null。

charAt、charCodeAt#

charAt(n) 返回字符串中的第n个字符，如果n超出字符串范围，返回空串。

charCodeAt(n) 返回字符串中第n个字符的字符编码。

includes、startsWith、endsWith#

includes(string,n) ES6 返回布尔值，表示是否找到参数字符串

startsWith() endsWith() 判断字符串开头、结尾 中间 有没有包含某个字符串

2、字符串截取、切割#

substr#

substr 根据指定长度截取字符串（不推荐使用该方法）

  • 第1个参数 起始下标
  • 第2个参数 截取的长度（如果省略，则代表截取到末尾）

slice、substring#

slice 、substring 根据指定的起止下标截取字符串

  • 第1个 起始下标
  • 第2个 结束下标（不包含），如果省略，代表剩下的全部截取
  • 起始和结束位置相比较的大小无法确定的时候可以使用substring
  • 如果为负值，slice能把负值当作从右向左。而substring视其无效

3、字符串转换#

valueOf 和 toString#

valueOf 返回对象的字符串、数值或者布尔值表示

toString 返回对象的字符串表示

split#

第一个参数如果是空串，则会分割每个字符

第二个参数用于指定分割后的数组大小，用来确保返回的数组不会超过这个大小

大小写转换#

toLowerCase() 将字符串转换成小写

toUpperCase() 将字符串转换成大写

padStart() 头部补全， padEnd() 尾部补全。

三、数组的基本使用#

数组（Array） 是一种引用类型，是有序的数据集合。它的**数组下标（index）**从0开始。数组中的每个成员被称为元素，不同元素的值可以为不同类型，数组的长度是弹性的，可读写的。

1. 创建数组#

（1）构造函数方式

var color = new Array('red','blue','green');
//  1. Array 如果仅一个元素且是数值，将创建数值个数元素的数组。
//  2. new可以省略。

（2）字面量方式（常用）

var color = ['red','blue','green'];

2. 数组的长度#

.length 属性保存数组的元素个数（长度）。可读可写，是一个动态属性，会随元素的变化而自动更新。

3. 数组基本操作#

var arr = [1,2,3,4,5,6];

// 1. 查看元素
arr[1]  // 查看第二个元素

// 2. 修改元素
arr[1] = 3

// 3. 在尾部加入一个元素（push方法替代）
arr[arr.length] = 7

4. 使用 for 遍历数组#

// 冒泡排序法
for ( let i=0; i<a.length-1; i++){
    for ( let j=0; j<a.length-1-i; j++){
        if (a[j] > a[j+1]){
            temp =a[j];
            a[j] =a[j+1];
            a[j+1] =temp;
        }
    }
}


// 斐波那契数列
// 打印出斐波那契数列的前100个值在数组中
// 1,2,3,5,8,13 初始值1和2   其他的值是前两个值的和

var arr = [1,2];
for (var i = 2; i < 100; i++) {
    arr[i] = arr[i-2] + arr[i-1];
}

5. 二维数组#

JS不支持多维数组，设置元素值为数组可以看成是一个二维数组。

6、扩展运算符#

扩展运算符（spread）是三个点（...）。它好比 rest 参数的逆运算，将一个数组转为用逗号分隔的参数序列，目前也可以用来展开数组。

7. 其他#

类数组 也称为伪数组，就是类似数组结构的对象。这个对象的属性名为非负整数，并且从0开始，有序递增，同时包含 length 属性，确保其值与有序下标属性个数保持一致，方便进行迭代操作。

// 遍历伪数组
[].forEach.call( arrayLike, function( item,index){
    //...
} )

// 转数组
[].slice.call( arrayLike )

空位数组 就是数组中包含空元素。所谓空元素，就是在语法上数组中两个逗号之间没有任何值。空元素可以读写，length属性不排斥空位，如果使用for语句和length属性遍历数组，空元素都可以被读取，空元素返回值为undefined。出现空位数组的情况如下:

  • 直接量定义
  • 构造函数定义
  • delete删除

关联数组 如果数组的下标值超出范围，如负数、浮点数、布尔值、对象或其他值，JS会自动把它转换为一个字符串，并定义为关联数组。关联数组就是与数组关联的对象，简单地说就是数组对象，字符串下标就是数组对象的属性。

var arr = [1,2];
arr[3.1] = "haha"; // 相当于给数组扩展了一些属性和方法
arr[true] = "buer";

四、数组方法#

1、栈和队列操作 △#

栈是一种先进后出的数据结构，队列是一种先进先出的数据结构。

.push() 接收任意数量参数，并逐个添加到数组末尾，返回修改后数组的长度。

.pop() 无参数，移除数组最后一项并返回该项。

.unshift() 接收任意数量参数，并逐个添加到数组头部，返回修改后数组的长度。

.shift() 无参数，移除数组第一项并返回该项。

这些方法都会修改原数组。

2、检索 △#

indexOf、lastIndexOf#

indexOf 返回的是下标，第二个参数是起始位置

indexOf 是全等查找，找不到NaN

includes △#

includes 返回一个布尔值，表示某个数组是否包含给定的值。

find#

find 找出符合条件的数组成员。

findIndex#

let result = arr.findIndex(fn)

传入一个测试函数，返回符合条件（测试函数返回ture时）数组的第一个元素位置，

如果没有符合条件则返回-1。

如果是空数组，测试函数是不会执行的，这个方法不会改变原先数组的值。

3、排序#

① 对当前数组进行排序，会修改当前数组。

② 参数可选，不传参数按默认unicode码进行排序，如果要进行数字的升序或者降序排序，需要传入一个比较函数作为参数。

// 升序
function fn(a, b){return a-b;}
// 降序
function fn(a, b){return b-a;}
// 按照数组对象中的某个属性进行排序
function fn(prop){
    return function(a, b){
        var v1 = a[prop];
        var v2 = b[prop];
        return v1-v2;
    }
}
// 冒泡排序法
for( var i=0; i<a.length-1; i++){
    for( var j=0; j<a.length-1-i; j++){
        // 正序
        if( a[j]>a[j+1]){
            t = a[j+1];
            a[j+1] = a[j];
            a[j] = t;
            
        }
    }
}

没有参数的情况下，该方法会调用每个数组元素的 toString 方法，然后比较的是字符串。在接收一个比较函数作为参数的情况下，来确定排序。

// v1
var res = values.sort(function (value1,value2)){
    if ( value1 < value2 ){
  			return -1;
		} else if (value1 > value2){
        return 1;
    } else {
      return 0;
    }
};


// v2 正序
var res = values.sort( function (value1,value2) ){
     return value1 - value2;
};

// 倒序
var res = values.sort( function (value1,value2) ){
     return value2 - value1;
};

4、删除、替换、插入、切割#

splice#

splice 可以对数组进行删除、替换、插入。

会修改原数组。

// （1）删除
//  第一个参数指定开始删除的index
//  第二个参数指定要删除的项数
arr.splice(1,2); 

//  (2) 替换
//  第一个参数指定开始index
//  第二个参数指定要删除的项数
//  第三个参数（或者更多的参数）指定要插入的对象
arr.splice(1,2,1,2);

//  (3) 插入
//  在（2）中，把第二个参数设置为0就可以插入内容了。

//  通过delete删除数组
let arr = [1,2,3,4];

delete arr[1];
arr[1];// undefined

slice#

slice 基于当前数组中的一个或者多个项目创建一个新数组，该方法不影响原始数组（实际上就是切割数组），一个参数就是从该位置切割到数组尾，两个参数就是切割起始位置和结束位置（不包括结束位置）。

var arr = ['a','b','c','d'];

var s1 = arr.slice(1);  // [ 'b', 'c', 'd' ]
var s2 = arr.slice(1,2); // ['b']
var s3 = arr.slice(1,3); // ['b','c']
var s4 = arr.slice();   // 和arr相同

7、forEach、map、filter、every、some#

（1）forEach 迭代数组

forEach 只能遍历数组。

不能使用break、continue退出。（可以使用 return）

// 过年了，每个人增加一岁
Arr.forEach( function( item, index, array){
    Arr [index] = item + 1;
})

// 没有返回值

（2）map 映射数组

map 被称为映射数组。参数是一个回调函数。该方法可以对数组的每一个元素调用指定的回调函数，并返回包含结果的数组。==返回一个新数组==，其中每一个元素均为关联的原始数组元素的回调函数返回值。不会为缺失的元素调用回调函数。

Arr. map( function( item, index){
    // 回调函数体
}, thisArg );
// 参数：
//   - item 项
//   - index 下标
// 返回值： 返回一个新数组

var newAge = oldAge.map( function(item, index,arr) ){
    return item + 1;
}

// 可以使用系统的内置函数

// 题目
[1, 2, 3].map[ parseInt ];   // [1,NaN,NaN]
// 调用了parseInt 进制转换方法
// 即 .map[ parseInt( item, index )]

（3）filter 过滤数组

返回数组中满足指定条件的元素。

Arr. filter (function, thisArg);

// 返回一个新数组

// 筛选出所有的偶数
var arr = [1,2,3,4,5,6,6,7,8,9];
var evenArr = arr.filter(function (item,index,array) {
    return item % 2 == 0
})
console.log(evenArr);//[2,4,6,6,8]

function f(item,index,array) {
    return item % 2 == 0;
}
var evenArr = arr.filter(f);
console.log(evenArr);


// 使用filter过滤掉不符合条件的值
var obj = {max:80,min:14};
var age = [12,11,34,56,90,92,65,65,34,31];
var newAge = age.filter(function (item) {
    return item >= this.min && item <= this.max;
},obj)
console.log(newAge);

（4）every 检测是否全部符合

  • every方法可以测试的所有元素是否都满足指定的测试
  • every中是函数，每一项数组的值，依次进入函数运行，函数最终要要求返回布尔值。
  • 如果有一项返回false，则停止执行，every返回false
  • 如果所有的项全部返回true 那么 整个every方法就返回true
  • every方法的回调函数拥有3个参数：1、item 数组的每一项 2、index 数组的这一项 对应的下标 3、array 原数组
  • every方法还有第二个可选的参数，可在回调函数中为其引用this关键字的对象。如果省略，则undefined是this值

（5）some 检测是否存在符合

  • 使用some方法可以确定数组的元素是否有满足指定的测试的项
  • 只要一项返回true 那么 some 方法就返回 true（一真即真）
  • 其余和every类似
// 我们得到一组年龄，如果所有的人都成年了，那么我们就开启成人网吧模式，否则就进入儿童模式
var userAge = [19,18,21,23,14,45];
var isPerson = userAge.some(function (item,index,array) {
    return item < 18;
});
if (isPerson){
    alert("儿童模式")
} else{
    alert("成人模式")
}


//this指向对象的
var arr = [18,19,20,21,30,11,14,19,40];
var isTrue = arr.some(function (item,index,array) {
    // return item < 18;
    return item < this.max;
},{max:18})
console.log(isTrue);
