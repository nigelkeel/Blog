一、显示#

1、display - 控制元素的显示模式 常#

控制元素的显示状态。

display 描述说明
inline 让元素以行属性显示
block 让元素以块属性显示
inline-block 让元素以行内块属性显示
none 带着所有子元素隐藏，直接消失
flex 开启弹性布局

display:none 隐藏的元素，不可以被 js 获取 DOM 节点及相关的样式属性。让隐藏的元素显示，只需要修改 display 属性的值不为 none 即可。

2、visibility - 控制元素显示或者隐藏#

visible: 元素显示
hidden: 元素隐藏
元素依然占用空间，和隐身一样，带着子元素一起隐身了。
子元素的隐藏是因为继承了父级的 hidden 值，所以单独给子元素设置，覆盖 hidden，即可显示。
使用 visiblity 隐藏的元素是可以用 js 获取的 DOM 节点及相应的宽高等属性。

visbility:hidden 和 display:none 隐藏的区别

①visibility 可以继承，子元素是因为继承了才隐藏（可覆盖）。display 不能被继承，直接带着所有内置元素直接隐藏。
②visbility 隐藏，保留原先占用空间。display 隐藏使原来的空间被占用。
③js 可以获取到 visibility 隐藏元素的可视化宽高，而获取不到 display 隐藏元素的可视化宽高。

3、overflow - 溢出设置 常#

内容溢出设置。

overflow 描述
visible 默认值
超出正常显示，但是不会影响其他元素的位置
hidden 超出部分隐藏
scroll 无论是否超出都生成滚动条
auto 自动生成滚动条，当内容溢出生成滚动条，否则不生成

overflow-x 和 overflow-y 可以单独控制一个方向的内容溢出设置。

隐藏横向滚动条：overflow-x: hidden

4、opacity - 透明度设置#

控制元素不透明度
值是 1-0
1 是完全不透明
0 是完全透明
opacity 直接控制所有的内部元素的不透明度，无论是什么元素都会改变

二、文本和字体样式#

01、color - 设置文本颜色#

（1）可设置颜色的类型

类型 示例 说明
英文名 red 开发中很少出现
RGB RGBA rgb(255,255,255)
十六进制 #112233 常用
HSL HSLA hsl(0,0,0) H 色调：0-360
S 饱和度 0-100%
L 亮度 0-100%
inherit 继承（默认值）
transparent 透明

（2）a 标签的 color 属性默认不是继承，需要的时候需直接控制。

02、font-family - 字体控制#

浏览器根据 font-family 解析渲染不同图形的过程 了解

  1. 读取文字内容转换成对应的 Unicode 码

  2. 根据 HTML 设置的 font-family （如果没设置使用浏览器默认设置）去查找到哪里对应的字体文件（如果有自定义字体 @font-face，则加载对应字体文件）

  3. 找到文件后根据 Unicode 码去查找绘制外形，找到后绘制到页面上

  4. 参考： 字体图标生成原理

03、@font-face - 自定义字体 常用#

04、font-size - 字体大小#

常用单位

  • px
  • em 参考父级的字号大小 前边的数字是倍数
  • rem 参考根元素的字号大小（谷歌浏览器默认字号大小 16px）

注：谷歌浏览器默认 16px，默认最小字号支持是 12px。

05、font-style - 字体风格#

  • normal 正常
  • italic 斜体
  • oblique 倾斜（火狐不支持）

06、font-weight - 字体粗细#

控制文字粗细。（可继承）

值 描述
normal 正常
bold 加粗
lighter 相对父级细一级
bolder 相对父级粗一级
100~300 细
400~500 正常
600~900 粗

07、font 合写#

font-style -weight -size/line-height -family
字号大小和字体必写，隐藏一个 line-height 属性，所以不继承的时候，需要检查 font 合写。

08、line-height - 行高控制 常用#

其实是控制两行文字基线的距离，决定了内联元素的占用的高度。默认会继承。

  • 固定值 px 不能是负
  • 比例：比如 1.5 就是当前文字大小的 1.5 倍
  • 百分比：80%
  • 浏览器默认值：normal

让单行文本垂直居中：行高 = 高度。

09、vertical-align - 垂直对齐方式#

控制某个行内元素在垂直方向的对齐方式。

baseline 基线（默认）
middle 中线
top 顶线
bottom 底线

图片垂直居中

10、text-align - 文本对齐设置#

text-align 文本对齐设置。控制当前元素内容。

属性值

  • left 默认值，左对齐
  • right 右对齐
  • center 居中对齐
  • justify 两端对齐

控制范围 无法让块元素居中

  • 元素内的文本
  • 行内块元素
  • 行元素

11、text-decoration - 文本修饰设置#

none 去掉下划线
underline 下划线
line-through 删除线
overline 上划线

线条的颜色是默认跟随文字颜色改变的。text-decoration 是间写属性，后边可以跟三个值：线的位置 线的颜色 线的类型。

12、text-indent - 文本缩进设置#

text-indent 文本缩进设置

属性值

  • 推荐单位为em， px不方便控制。
  • 可以为负，实现悬挂缩进的效果。

控制范围

  • 只针对块元素生效

13、letter-spacing、word-spacing - 间距控制#

letter-spacing
控制字符间距
控制所有字符和字符之间的间距

word-spacing
控制词间距
改变词和词之间的间距
以空格来分割两个词 值都可以为负

14、white-space - 空白处理#

值 描述
normal 默认。空白会被浏览器忽略。
pre 空白会被浏览器保留。
其行为方式类似 HTML 中的 <pre> 标签。
nowrap 文本不会换行，文本会在在同一行上继续，直到遇到 <br> 标签为止。
pre-wrap 保留空白符序列，但是正常地进行换行。
pre-line 合并空白符序列，但是保留换行符。
inherit 继承

white-space 详解

三、背景样式#

任何背景是撑不开元素的宽高。

1、background-color - 背景颜色#

背景颜色是从边框下开始显示的。

设置值同 color。

2、backgroud-image - 背景图片#

background-image: url(路径)
边框下也显示图片，但是是平铺出来的。
背景图片的左上角原点，是以 padding 区域的左上角开始显示的。

多重背景 CSS3#

CSS3 允许为元素使用多个背景图片。

元素引入多个背景图片，前面图片会覆盖后面的图片。

3、background-position - 背景图定位#

由两个值构成（background-position-x background-position-y），值可以是 （坐标轴是以 padding 区域的左上角原点的坐标）

  1. px
  2. 关键字
    在 x 轴上有 left right center
    在 y 轴上有 top bottom center
  3. 百分比
    当百分百的时候，和 right bottom 效果一致

4、background-repeat - 背景图平铺#

repeat 默认 x 和 y 都平铺
repeat-x 只沿着 x 轴平铺
repeat-y 只沿着 y 轴平铺
no-repeat 不平铺

5、background 合写#

background: color image position-x position-y repeat
如果有 background 合写属性，那么当此元素还有关于 background 属性的时候，请书写在合写的下边。

6、background-origin - 背景图原点#

background-origin: padding-box (默认)

  • border-box | padding-box | content-box
  • background-origin 是用来决定图片的原始起始位置。
  • 它有三个可选值 content-box,padding- box,border-box，即可以选择背景图片是从内容区域或者内边距或者边框开始显示。

7、background-clip - 背景图裁剪#

background-clip

  • border-box | padding-box | content-box
  • background-clip 的作用为将背景图片做适当的裁剪，以适应需要。
  • background-clip 有 content- box,padding-box,border-box，text 几个值剪裁方法：根据设置的盒子部位，那么图片在这个部位的外边缘以外的部分都会不可见。

8、background-size - 背景图大小#

background-size

  • length： 长度值—第一个值设置宽度，第二个值设置高度
  • percentage： 百分比—第一个值设置宽度，第二个值设置高度
  • cover：充满容器，比例不发生变化，但是有可能超出。
  • contain： 将背景图像等比缩放到宽度或高度与容器的宽度或高度相等，背景图像始终被包含在容器内

9、精灵图#

CSS Sprites，精灵图，是网页图片的常用处理方式。

允许将一个页面中很多零星的小图片包含到一张大图中去，当访问页面的时候，就不会一张张的去请求显示图片。对网络来说，小于 200k 的文件加载速度是一样的。通过 background-position 属性，将图片定位到需要的位置即可。

优点：

  • 减少图片大小
  • 减少服务器请求次数

6、渐变#

线性渐变 linear-gradient#

  • 从上到下（默认）：background: linear-gradient(red,blue);
  • 从左到右：background: linear-gradient(to right,red,blue);
  • 对角：background: linear-gradient(to right bottom,red,blue);
  • 角度：background: linear-gradient(45deg,red,blue);

径向渐变 radial-gradient#

  • 第一个参数：圆的类型
  • ellipse：根据元素形状渐变，元素为正方形是显示效果与 circle 无异
  • circle：渐变为最大的圆形
  • 第二个参数：执行径向渐变的大小
  • farthest-corner:圆心到最远角
  • closest-side：圆心到最近的边
  • closest-corner：圆心到最近的角
  • farthest-side：圆心到最远的边
  • 第三个参数：圆心的位置
  • 第四个参数：渐变的颜色点

五、杂项#

1、样式操作#

样式重置（CSS reset），将 html 的默认样式全部去掉，需要的时候我们自行添加。统一页面风格。

为什么要用样式重置

  • 多数元素拥有自己的默认样式，并且我们很多都不需要。
  • 默认样式在不同浏览器中呈现的也不一定相同，就会导致浏览器展示页面不相同的现象。
  • 整个页面中固定的风格代码，可以在样式重置中直接书写，不用每一个都设置。

为什么不用通配符重置

  • 重置简单，不够完整，一些 demo 可以使用，真正项目不推荐使用，请使用 CSS reset
  • 通配符，匹配所有的标签，影响很大，效率很低。
