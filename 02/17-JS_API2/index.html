一、JSON#

JSON 是ES内置对象（不用定义就能用）。

json 是一种常用于前后端数据交互的格式，本质是字符串。

1、值#

可以表示三种类型数据：

（1）简单值

除了 undefined 之外的基本数据类型，表示字符串必须使用双引号。

5
"hello world"   //表示字符串需加双引号
null
false

（2）对象 表示的是一组无序的键值对，属性必须用双引号包起来。

{
    "name":"Tom",
    "age""18
}

（3）数组 有序的集合、键值对。

[25, "hi", true]

2、方法#

用于前后台发送数据。

（1）JSON.stringify()#

将JS值序列化为JSON字符串。

函数，原型成员，undefined会被忽略。

（2）JSON.parse()#

将JSON字符串解析成原生的JS值。

不是有效的JSON字符串就会报错。

二、Date#

Date 是JS中用来处理日期和时间的内置对象。

参考链接

1、基本使用#

获取一个Date对象

let now = new Date();
now // 2020-08-26T12:11:25.743Z
typeof now // object

创建一个时间

  • 传入一个毫秒数（会把毫秒数转换成一个时间c 然后加上1970年1月1日 8：00 的时间）
  • 传入一个字符串格式的时间
  • 传入多个参数 分别代表年 月 日 时 分 秒 毫秒
var date1 = new Date(1000 * 60 * 60);
var date1 = new Date(1546354578234);
console.log(date1);

var date2 = new Date("2019-10-01 8:0:0");
var date2 = new Date("2019-10-1");//如果不写时间 只写年月日 那么时间按照00点来计算
var date2 = new Date("8:0:0");//Invalid Date 如果不写年月日 那么时间是错误的
console.log(date2);

//当数字形式传递或者获取月份的时候，月份是从0开始算  0 代表 1月
var date3 = new Date(2019,9,1,8,10,20,300);
//时间超出分为 会自动向前进一位  但是不建议这么写
var date3 = new Date(2019,16,1,8,10,20,300);
console.log(date3);

2、时间戳#

时间戳是一个整数，保存1970年1月1日到现在的毫秒数。

获取时间戳方法

  • Date.now()
  • new Date().getTime()
  • Date.parse(new Date())
  • new Date().valueOf()

3、设置时间#

// 获取当前的时间戳
Date.now();

// 获取当前时间
let now = new Date();
// now => 2020-07-14T06:00:41.645Z

// 作为构造函数使用
let dt  = new Date(年,月,日,[时,分,秒]);

自动校准机制，超过范围的会自动换算。

4、时间获取#

方法 功能 范围
getFullYear() 获得四位年份
getMonth() 获取月份 0-11
getDate() 获取月份中的天数 1-31
getDay() 获取星期，0表示周日 0-6
getHours() 小时 0-23
getMinutes() 分钟 0-59
getSeconds() 秒数 0-59
getMilliseconds() 毫秒数 0-999

5、设置单独的时间#

  • 设置年份 setFullYear
  • 设置月份 setMonth
  • 设置日 setDate
  • 设置小时 setHours
  • 设置分钟 setMinutes
  • 设置秒 setSeconds
  • 设置毫秒 setMilliseconds
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>设置时间</title>
</head>
<body>
<script>
    var now = new Date();

    //设置年份
    now.setFullYear(2023);
    console.log(now);

    //设置月份
    now.setMonth(0);
    console.log(now);

    //设置日期
    now.setDate(4);
    console.log(now);

    //设置 小时
    now.setHours(19);
    console.log(now);

    //设置分钟
    now.setMinutes(40);
    console.log(now);

    //设置秒
    now.setSeconds(10);
    console.log(now);

    //设置毫秒
    now.setMilliseconds(100);
    console.log(now)

    //设置星期  不允许 没有这个方法
    now.setDay();
    console.log(now);
</script>
</body>
</html>

6、示例#

// 获取类似 2020-08-08 12:04:56 格式的字符串
function () {
    const time = new Date();
    return time.getFullYear() + "-" +
        (time.getMonth() + 1).toString().padStart(2, "0") + "-" +
        time.getDate().toString().padStart(2, "0") + " " +
        time.getHours().toString().padStart(2, "0") + ":" +
        time.getMinutes().toString().padStart(2, "0") + ":" +
        time.getSeconds().toString().padStart(2, "0");
}

三、Math#

1、进制#

（1）进制表示

// 二进制

2、取整#

将小数转换成整数，如果是整数则不改变。

  • Math.ceil() 进一取整
  • Math.floor() 舍一取整
  • Math.round() 标准的四舍五入

3、最值#

Math.max() Math.min() 获取一个参数列表的最大值，最小值。

// 获取一个数组的最值
let arr = [1,2,3,4,5,6,7]

// ES5
Math.max.apply(null,arr);

// ES6
Math.max(...arr);

7、生成唯一的随机字符串#

function uniqueId() {
  // substring去掉.
  const randomNum = Number(Math.random().toString().substring(2));
  // .toString(32) 转换为32进制
  return (randomNum + Date.now()).toString(32);
}
