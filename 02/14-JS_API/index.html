一、字符串 API#

除非特别标注，则默认不会改变原字符串的。

查找 - indexOf / lastIndexOf#

indexOf() 返回指定字符串在字符串首次出现的位置（向后查找，对大小写敏感），如果没找到，返回-1。

STR. indexOf(search [, start])
// 参数
//    search 需检索的字符串
//    start  可选，指定开始检索的位置
//          如果没指定就是从首字符开始搜索
// 返回值
//    目标字符串第一次出现的位置/-1

let str = “poxiaoispig";
str.indexOf("poxiao"); // 0
str.indexOf("pig"); // 8

lastIndexOf() 同 indexOf() ，区别是逆序的（向前查找）。

str.lastIndexOf("pig"); // 8

查找 - includes#

includes() 查找字符串是否包含指定字符串（对大小写敏感），如果找到返回true ,否则返回false。

STR. includes(search, [, start])
// 参数
//    search 需查找的字符串
//    start  可选，指定开始检索的位置
//          如果没指定就是从最后一个字符开始搜索
// 返回值
//    目标字符串第一次出现的位置/-1

let str = “poxiaoispig";
str.includes("poxiao"); // true
str.includes("px");   // false

查找 - search#

search() 功能和indexOf()相同，定义匹配模式，只有一个参数，没有反向检索和全局模式。（正则表达式常用）

STR. search(tar)
// 参数
//    tar 要搜索的字符串
// 返回值
//    目标字符串第一次出现的位置/-1

let str = "poxiaoispig";
str.search("pig");  // 8

查找 - match#

match() 找出所有匹配的子字符串，并以数组的形式返回，如果匹配模式没有指定全局（正则表达式），则进行单次匹配，如果没找到返回null。该方法类似 indexOf() 和 lastIndexOf()，但是它返回指定的值，而不是字符串的位置。

STR. match(re)
// 参数
//    re  字符串或者正则表达式
// 返回
//    数组（包含所有匹配的字符串）

let str = "poxiao";
str.match("xiao"); 
// [ 'xiao', index: 2, input: 'poxiao', groups: undefined ]

查找 - charAt/charCodeAt#

charAt(n) 返回字符串中的第n个字符，如果n超出字符串范围，返回空串。

charCodeAt(n) 返回字符串中第n个字符的字符编码。

判断头尾 - startsWith/endsWith#

startsWith() endsWith() 判断字符串开头、结尾有没有包含某个字符串。

STR. startsWith(key)
STR. endsWith(key)
// 参数
//    key 目标字符串
// 返回
//    true / false

"poxiao".startsWith("po"); // true
"poxiao".endsWith("po");  // false

切成数组 - split#

split() 把一个字符串切割成数组，这个操作和数组join是相反的。

STR. split(rule[, max])
// 参数
//    rule 字符串或者正则表达式
//    max  可选，指定返回数组的的最大长度
// 返回值
//    数组

let str = "poxiao";
str.split(); // ["poxiao", ]
str.split(""); // ["p","o","x","i","a","o"]
str.split("", 2); // ["p","o"]

截取 - substr/slice/substring#

substr 根据指定长度截取字符串（不推荐使用该方法）

STR. substr(start[, length]);
// 参数
//    start  起始下标
//    length 可选，截取的长度
//           如果省略，则截取到末尾
// 返回值
//    截取之后的字符串

slice 、substring 根据指定的起止下标截取字符串

STR. slice(start[, end]);
STR. substring(start[, end]);
// 参数
//    start 起始下标
//    end   可选，结束下标（不包括）
//          如果省略，代表剩下的全部截取

起始和结束位置相比较的大小无法确定的时候可以使用substring。

如果为负值，slice能把负值当作从右向左。而substring视其无效

大小写转换 - toLowerCase / toUpperCase#

toLowerCase() 将字符串转换成小写

toUpperCase() 将字符串转换成大写

STR. toLowerCase()
STR. toUpperCase()
// 参数无
// 返回
//     转换之后的字符串

let str = "poxiao";
str.toLowerCase();//"POXIAO"

连接 - concat#

concat() 用于连接字符串。

STR. concat(str1[, str2, str3...])
// 参数
//    str 要拼接的字符串（一个或者多个）
// 返回值
//    连接之后的字符串。

let str = "poxiao";
str.concat("is", "pig"); // "poxiaoispig"

复制 - repeat#

repeat() 复制字符串。

STR. repeat(count);
// 参数
//    count 复制次数
// 返回值
//    复制之后的字符串

let str = "poxiao";
str.repeat(3); // "poxiaopoxiaopoxiao"

补全 - padStart / padEnd#

padStart() padEnd() 是 ES2017 引入的字符串补全长度的方法。如果某个字符串不够指定长度，会在头部或尾部补全。

STR. padSrart(length, pad)
STR. padEnd(length, pad)
// 参数
//    length 指定字符串的最小长度
//    pad    用来补全的字符串
// 返回值
//    补全之后的字符串
//    如果原字符串的长度，等于或大于指定的最小长度，则返回原字符串。

let str = "poxiao";
str.padStart(2, "1"); // "poxiao"
str.padStart(10, "is"); // "isispoxiao"

替换 - replace#

如果没有使用正则表达式，只能替换一次。

这是用于搜索和替换的通用方法，是最有用的方法之一。它是搜索和替换字符串的瑞士军刀。

当 replace 的第一个参数是字符串时，它仅替换第一个匹配项。

删首尾空格 - trim / trimStart / trimEnd#

trim() trimStart() trimEnd() 去除字符串的头尾空格。

STR. trim()
STR. trimStart()
STR. trimEnd()
// 无参数
// 返回处理之后的字符串（不会改变原字符串）

let str = "   poxiao is pig   ";
str.trim() // "poxiao is pig"
str.trimStart() // "poxiao is pig   "
str.trimEnd()  // "   poxiao is pig"

转换 - valueOf / toString#

valueOf() 返回对象的字符串、数值或者布尔值表示。

toString() 返回对象的字符串表示。

二、数组 API#

如果没有特别说明，则说明这个API不会修改原数组。

长度 - length#

length 属性保存数组的元素个数（长度）。可读可写，是一个动态属性，会随数组的变化而自动更新。

栈/队列操作 - push/pop/unshift/shift#

这些方法都会修改原数组。

push() 接收任意数量参数，并逐个添加到数组末尾，返回修改后数组的长度。

pop() 无参数，移除数组最后一项并返回该项。

unshift() 接收任意数量参数，并逐个添加到数组头部，返回修改后的数组的长度。

shift() 无参数，移除数组首项并返回该项。

查找 - indexOf/lastIndexOf#

indexOf() 返回的是下标，第二个参数是起始位置。

这个方法是全等查找，找不到 NaN。

查找 - includes#

includes() 返回一个布尔值，表示某个数组是否包含给定的值。

查找 - find/findIndex#

find() 找出符合条件的数组成员。

findIndex() 传入一个测试函数，当测试函数返回 true 时，返回数组的一个元素位置。如果没有符合条件返回 -1。

如果是空数组，测试函数是不会执行的。

排序 - sort △#

sort() 对当前数组进行排序，会修改当前数组。

参数是一个比较函数，如果不传入参数，该方法会调用每个数组元素的 toString() 按默认 unicode 码进行排序。

Array. sort ();

// 例子
[1, 3, 2, 4, 6, 5].sort((v1,v2)=>v1-v2); // [ 1, 2, 3, 4, 5, 6 ]
[1, 3, 2, 4, 6, 5].sort((v1,v2)=>v2-v1); // [ 6, 5, 4, 3, 2, 1 ]

// 依据对象数组中的某个属性进行排序
let data = [
    {id:3, name:"水"},
    {id:1, name:"金"},
    {id:5, name:"土"},
    {id:4, name:"火"},
    {id:2, name:"木"}
]
const order = (prop) => (a, b) => a[prop] - b[prop];
data.sort(order("id"));

操作 - splice/slice △#

splice() 可以对数组进行删除、替换、插入，这个方法会修改原数组。

Array. splice(start, count[, insert, ...])
// 参数
//     start  开始删除的index
//     count  要删除的项数
//     insert 要插入的内容
// 返回
//     数组（被删除的内容）


// 删除
let arr1 = [1, 2, 3, 4, 5];
arr1.splice(1, 2);// 从index=1的地方开始，删除2项 // [2, 3]
console.log(arr1); // [1, 4, 5]

// 替换
let arr2 = [1, 2, 3, 4, 5];
arr2.splice(1, 2, 666, 777); // [2, 3]
console.log(arr2); // [1, 666, 777, 4, 5]

// 插入
let arr3 = [1, 2, 3, 4, 5];
arr3.splice(1, 0, 666); // []
console.log(arr3); // [1, 666, 2, 3, 4, 5]
