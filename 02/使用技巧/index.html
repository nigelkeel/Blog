使用技巧

发表于 2020-08-27 分类于 技术栈加

一、Windows 10#

  • 在地址栏 输入 cmd powershell 可以在当前目录打开命令行。
  • Win + Ctrl + ←/→ 切换虚拟桌面
  • CTRL + SHIFT + V 无格式黏贴

PotPlayer#

快捷键 功能
X 加速
C 减速

编辑器 通用#

  • 搜索：ctrl + F
  • 替换：ctrl + H
  • 选择一整行：CTRL + L

VSCode#

  • 向下复制当前行：SHIFT + ALT + ↓
  • 注释和取消注释：CTRL + /
  • 多行注释：CTRL + shift + /

Typora#

  • 源代码模式：CTRL + /
  • 打字机模式：F8
  • 专注模式：F9
  • 显示/隐藏侧边栏：CTRL + SHIFT + L
  • 段落：CTRL + 0
  • 标题：CTRL + 1/6
  • 表格：CTRL + T
  • 选中一整行：CTRL + L
  • 选中单词：CTRL + D
  • 跳转到文章开头：CTRL + HOME
  • 跳转到文章结尾：CTRL + END
  • 插入链接：CTRL + K
  • 设置：CTRL + ，
  • 返回顶部: CTRL + Home
