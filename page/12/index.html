Windows 10#

  • electron打包踩过的坑总结

MacOS 打包环境配置#

推荐系统版本：MacOS 10.14，经过测试，10.15 会出现兼容性问题

  1. 通过 SVN 工具 checkout 代码到本地

    推荐用 Cornerstone

  2. 在官网下载并安装 Node.js

    推荐版本：8.12

    也可以安装最新稳定版本，然后通过 npm 版本切换工具 n 来切换版本

    sudo npm i n -g

    sudo n 8.12

    sudo n 选择相应的版本回车即可

  3. 全局安装 electron-builder

    sudo npm i electron-builder@20.19.2 -g

    另外如果 20.19.2 不行可以换 20.28.4 试试

  4. 终端进入要打包的版本目录，安装依赖

    cd 你的目录

    sudo npm i -unsafe-perm

    注：如果没有科学上网的话，下载 electron 会非常慢

  5. 安装完依赖执行打包命令

    sudo npm run build:mac

    注：这里需要科学上网，不然可能卡在 downloading 地方

    如果失败，请进入资源库/Cache/Electron 删除全部内容，重新尝试打包

    科学上网只要第一次就可以了，下次打包会有缓存

  6. 下次打包之前运行

    sudo npm run build:clean

  7. 如果出现问题，可以尝试清 npm 缓存

    sudo npm cache verify

    然后删除版本目录下面的 node_modules ，重新 sudo npm i --unsafe-perm 安装依赖

平面设计基本原则#

  • 对齐：可用网格、辅助线。
  • 重复：控制元素数量，避免杀马特化。
  • 对比：重复太多需要用对比来强调。
  • 亲密性：物理位置的接近意味着内容的关联。

扁平化#

扁平化的概念最核心的地方就是：去掉冗余的装饰效果，去掉多余的透视，纹理，渐变等等能做出3D效果的元素。让信息本身重新作为核心被凸显出来。并且在设计元素上强调抽象、极简、符号化。

1、链接的文件名字带空格不识别怎么办？#

解决: 在链接中用%20替代空格的书写

2、永久插入图片#

首先将图片通过在线转码工具转换成base64的编码，并用以下格式嵌入即可，格式如下：

![image] (base64)

但是由于这base64编码往往都很长，很占篇幅，因此我们可以给图片编号，并将所有的图片base64编码放在文档的最后即可，格式如下：

  • 在插入图片的地方使用：![image] [图片编号]
  • 在文档最后使用：[图片编号]:base64编码

这样即可完成图片的永久插入！

发表于 2021-01-12 更新于 2021-02-18 分类于 2202：ES高级
本文字数： 288

1、什么是闭包#

闭包是一个包含被引用局部变量的 closure 对象，存在嵌套的内部函数中。

2、怎么产生闭包 & 生命周期#

必须满足三个条件

  • 函数嵌套函数
  • 内部函数调用外部函数的局部变量（引入全局变量是不会产生闭包的）
  • 调用外部函数

生命周期

  • 内部函数定义执行完成就产生
  • 内部函数成为垃圾对象就销毁

3、闭包的作用与缺点#

作用

  • 延长了局部变量的存活时间
  • 让函数外部操作函数内部的数据

缺点

  • 用太多可能会导致内存泄露（因为占用内部空间不释放）

4、闭包的应用#

  • 迭代器
  • React 组件高阶函数（为了复用函数，给内部函数传参）
  • Vue 原理中存在（双向数据绑定原理）
