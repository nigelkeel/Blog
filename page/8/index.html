Anaconda#

一、安装和配置环境变量#

1、参考：Anaconda详细安装及使用教程
2、查看版本 conda --version
3、升级工具包 conda upgrade --all

二、管理虚拟环境#

1、创建虚拟环境#

创建一个名称为test的虚拟环境并指定 Python 版本为3.7。

conda  create -n test python=3.7

2、切换环境#

切换到base环境

activate

切换到名叫test的环境。

activate test

查看所有环境

conda env list

3、卸载环境#

卸载名叫test的环境。

conda remove --name test --all

4、安装和卸载第三方包#

安装第三方名叫test的包。

conda install test

或者

pip install test

卸载第三方名叫test的包。

conda remove test

或者

pip uninstall test

查看所有包列表

conda list

三、问题#

1、解决Win10 PowerShell无法激活Anaconda环境的问题

2、更改PowerShell执行策略

1. 以管理员身份运行PowerShell;
2. 执行：get-ExecutionPolicy，显示Restricted，表示状态是禁止的;
3. 执行：set-ExecutionPolicy RemoteSigned;
4. 这时再执行get-ExecutionPolicy，就显示RemoteSigned;

Python Random#

随机数库

1、引入

import random

2、生成一个指定范围内的随机浮点数

random.uniform(n1,n2)

3、生成一个指定范围内的整数

random.randint(n1,n2)

4、从指定范围内，按指定的基数递增的集合中获取一个随机数

random.randrange([start],stop[,step])

5、从序列中获取一个随机元素

random.choice(序列)

6、打乱列表元素

random.shuffle(x[,random])

7、从序列中随机获取指定长度的片段

random.sample(n1,n2)

sqlite#

with Python

1、引用

import sqlite3

2、连接或者创建数据库，如果数据库不存在将在工作目录创建数据库

conn = sqlite3.connect('数据库名称.db')

3、获取游标

c = conn.cursor()

4、新建表

c.execute('''
       CREATE TABLE 表名(
       ID INT PRIMARY KEY NOT NULL,
       NAME TEXT NOT NULL,
       AGE INT NOT NULL,
       ADDRESS CHAR(50),
       SALARY REAL
       );
''')

5、插入操作

c.execute("INSERT INTO 表名 (key1,key2) VALUES(value1,value2);")

6、查询操作

7、更新操作

8、删除操作

9、提交事务

Python 的 SQLite 数据库 API 默认是开启了事务的，因此需提交事务；否则程序对数据库所做的修改（包括插入数据、修改数据、删除数据）不会生效。

conn.commit()

10、关闭游标和连接

c.close()
conn.close()

学习#

一、学习方式#

1、前提#

学习首先是自信心，利用20小时就可以学习一门大学教材。

2、学习的过程#

学习分为预学习和正式学习。

2.1 预学习#

预学习，开始学习的学习仪式感。
预备，确定学习主题。
预热，物质准备，精神准备，环境准备。
静静等待。

2.2 正式学习#

  • 初阅：
    看目录，大概了解。
    看课后习题，术语 = 知识点。
    配合最少量的文字看懂案例、图表，就达到了对知识的了解。
  • 再阅：正式学习，补充，偏差修正。
  • 回顾：刚刚主要学了什么？
  • 练习：刻意练习的重要性。
  • 测试与检验。

3、学习技巧#

  • 高潮停顿的技巧。
  • 惯性原理：静止的倾向于静止，运动的倾向于运动。

4、注意事项#

  • 注意需要很好的放松，不然一段时间后会有瓶颈和临界状态。
  • 能量分为身体能量和情绪能量。情绪能量低，自控力也低。如何恢复情绪能量：放松。
  • 注意倦怠感的释放，它会积累。

二、学习的观点#

  • 工具不能代替学习。
    工具应该是用来提供帮助的，用来帮助防止已知错误的，并帮助我们记住重复的任务，而不是取代思考。
  • 学习的必要经过是犯错误。
  • 学习或研究一门新的知识时，最好的办法就是去学习这一方面成功人士的方法或系统，然后根据自己的情况进行调整，形成自己的系统，再不断实践并改进。
  • 与其花费一万小时，不如在重难点上花费一百小时。
  • 学习与体能的关系：更好的体能等于更好的注意力。

建立知识体系#

一、开始#

为什么要建立知识体系？

  • 大脑的工作机制是基于索引
  • 学那么多东西，为什么记不住，用不上？是因为你学到的都是零碎的，不成体系的。
  • 我们给知识建立体系就是给知识建立索引，以及索引间的逻辑关系，当知识体系大厦搭建完毕以后，你就可以随意调用，只要在你知识体系涵盖的范围内，就可以任意使用。

二、如何建立知识体系#

构造知识体系的材料#

（1）元认知（地基）

  • 能指导你思考背后的思考，指导你解释原理背后的原理。
  • 最底层的知识，它为一切现象原理，提供最底层的解释和支撑。

（2）方法（钢筋水泥）

  • 方法，套路。

（3）概念（砖瓦）

  • 我们身处的世界都是由一个个概念构成的，这个时候我们理解事物的基本方式。
  • 可以说我们认知这个世界的方式，就是在认知一个个概念，概念是组成我们思想的基本方式，你了解的概念越多，你认识这个世界的维度就会越丰富。

（4）事实（边料）

  • 浩如烟海。
  • 这些知识只在特定情况有用，大多数情况下是没有用的，所以它的可迁移能力最弱。

知识的可迁移#

  • 知识的可迁移能力，就是我们学习的任何东西，都是为了实现知识的最大可迁移，我们可以从一个领域，迅速用同样的方式，进行跨界。
  • 元认知类知识最有用，迁移能力最强，但学习难度最高，而事实类知识迁移能力最弱，学习难度也是最容易的。

  • 无用之用是为大用

  • 让知识间建立关系，切记一定不要孤立

目标管理#

别去管目标：
减肥20磅是一个目标，而正确饮食则是一个体系。
在4个小时内跑一次马拉松是一个目标，而每日锻炼则是一个体系。
体系指的是您定期做的某件从长远来看能够增加您幸福几率的事情。
—— Scott Adams

设置目标的原则#

目标不可贪多。
目标太多，什么事都做不好，做不全。你就会开始怀疑自己，直至放弃。刚开始列出我的一周计划时候，满满的一张纸，看着这么多的目标，感觉自己真的很有激情，为自己每一刻都在努力而喜悦。结果刚开始两天，就败下阵来。后来试着删除简化我的目标清单，就会发现，目标瞄准一两件事后，越来越有自信并且越有工作状态，心有余力时，我再去想着给自己安排另外自己想做的事情。推荐利用类似四象限法思维进行思考，按照紧急重要、重要不紧急、紧急不重要、与不重要不紧急分级安排事件。

分解任务，适当休息。
很多工作没有办法在一定时间内完成时，就会使人疲惫抑或放弃，所以懂得如何分解任务就显得非常必要了。利用番茄工作法，你可以延长专注也可缩短休息，但不可不休息，人的生理状态与记忆能力等都需要适当的缓冲而非持续工作。

关键性目标
关键性目标是指你的投入与回报不对等，有些习惯只需要很少的投入就能带很大的回报。比如说，每天10-30分钟的有效运动，只要坚持2、3个月，给你带来的各种效果会超出你的想象。从以往的100天行动反馈中来看，运动、冥想、慢跑、每天优先完成三件要事、一个梦想番茄、不抱怨、赞美、保持冷静、存入收入的10%、定期回顾、永远不说谎、保持冷静和放松，这些习惯的回报要比其它的习惯高一些。

列出ToDoList#

待办事项清单可以分为两类，一类是long-term list，一类是short-term list。long-term list 可以以一周一个月一年来记录，也就是说你所有做的一切都是朝着这个目标前进的，你列出清单也是为了更好的审视自己是否完成是否偏离。short-term list这个也主要分为两类，一类是你生活琐事清单，一类是你的目标清单。生活琐事就如打电话，买水果，洗衣服等等。目标清单就是朝着long-term list 的一步步脚印。
