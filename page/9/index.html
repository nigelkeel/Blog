ES Set#

ES6+

一种数据结构，类似数组，但是成员的值都是唯一的，不能重复。

let set = new Set([1, 2, 2, 3]);
console.log(set); // Set {1, 2, 3}

API#

api 描述
size 返回 Set 的长度
add() 添加一个值，返回 Set 结构本身
delete() 删除一个值，返回一个布尔值
has() 判断是否存在某成员，返回一个布尔值
clear() 清除所有成员
keys() values 遍历键值，同 for..of
entries() 遍历返回键值对
forEach() 使用回调函数遍历每个成员

应用#

去除数组重复成员

[...new Set(array)];

去除字符串里面重复字符

[...new Set(str)], join("");

字符串 API#

字符串是基本类型，所有的方法都无法修改原字符串。

查找 - indexOf / lastIndexOf#

indexOf() 返回指定字符串在字符串首次出现的位置（向后查找，对大小写敏感），如果没找到，返回-1。

STR. indexOf(search [, start])
// 参数
//    search 需检索的字符串
//    start  可选，指定开始检索的位置
//          如果没指定就是从首字符开始搜索
// 返回值
//    目标字符串第一次出现的位置/-1

let str = “poxiaoispig";
str.indexOf("poxiao"); // 0
str.indexOf("pig"); // 8

lastIndexOf() 同 indexOf() ，区别是逆序的（向前查找）。

str.lastIndexOf("pig"); // 8

查找 - includes#

includes() 查找字符串是否包含指定字符串（对大小写敏感），如果找到返回true ,否则返回false。

STR. includes(search, [, start])
// 参数
//    search 需查找的字符串
//    start  可选，指定开始检索的位置
//          如果没指定就是从最后一个字符开始搜索
// 返回值
//    目标字符串第一次出现的位置/-1

let str = “poxiaoispig";
str.includes("poxiao"); // true
str.includes("px");   // false

查找 - search#

search()功能和indexOf()相同，定义匹配模式，只有一个参数，没有反向检索和全局模式。（正则表达式常用）

STR.search(tar);
// 参数
//    tar 要搜索的字符串
// 返回值
//    目标字符串第一次出现的位置/-1

let str = "poxiaoispig";
str.search("pig"); // 8

查找 - match#

match() 找出所有匹配的子字符串，并以数组的形式返回，如果匹配模式没有指定全局（正则表达式），则进行单次匹配，如果没找到返回null。该方法类似 indexOf() 和 lastIndexOf()，但是它返回指定的值，而不是字符串的位置。

STR.match(re);
// 参数
//    re  字符串或者正则表达式
// 返回
//    数组（包含所有匹配的字符串）

let str = "poxiao";
str.match("xiao");
// [ 'xiao', index: 2, input: 'poxiao', groups: undefined ]

查找 - charAt/charCodeAt#

charAt(n) 返回字符串中的第 n 个字符，如果 n 超出字符串范围，返回空串。

charCodeAt(n) 返回字符串中第 n 个字符的字符编码。

判断头尾 - startsWith/endsWith#

startsWith() endsWith() 判断字符串开头、结尾有没有包含某个字符串。

STR.startsWith(key);
STR.endsWith(key);
// 参数
//    key 目标字符串
// 返回
//    true / false

"poxiao".startsWith("po"); // true
"poxiao".endsWith("po"); // false

切成数组 - split#

split() 把一个字符串切割成数组，这个操作和数组join是相反的。

STR. split(rule[, max])
// 参数
//    rule 字符串或者正则表达式
//    max  可选，指定返回数组的的最大长度
// 返回值
//    数组

let str = "poxiao";
str.split(); // ["poxiao", ]
str.split(""); // ["p","o","x","i","a","o"]
str.split("", 2); // ["p","o"]

截取 - substr/slice/substring#

substr 根据指定长度截取字符串（不推荐使用该方法）

STR. substr(start[, length]);
// 参数
//    start  起始下标
//    length 可选，截取的长度
//           如果省略，则截取到末尾
// 返回值
//    截取之后的字符串

slice 、substring 根据指定的起止下标截取字符串

STR. slice(start[, end]);
STR. substring(start[, end]);
// 参数
//    start 起始下标
//    end   可选，结束下标（不包括）
//          如果省略，代表剩下的全部截取

起始和结束位置相比较的大小无法确定的时候可以使用substring。

如果为负值，slice 能把负值当作从右向左。而substring视其无效

大小写转换 - toLowerCase / toUpperCase#

toLowerCase() 将字符串转换成小写

toUpperCase() 将字符串转换成大写

STR.toLowerCase();
STR.toUpperCase();
// 参数无
// 返回
//     转换之后的字符串

let str = "poxiao";
str.toLowerCase(); //"POXIAO"

拼接 - concat 了解#

concat(s1[, s2, s3])

功能：把多个参数添加到指定字符串的尾部。

参数：要添加到字符串尾部的，参数类型和个数没有限制。

返回：连接以后的字符串。

let str = "poxiao";
str.concat("is", "pig"); // "poxiaoispig"

原来数组的 join 方法替代 concat 方法，速度很快，但是现代浏览器优化加号拼接，所以推荐使用加号拼接。

复制 - repeat#

repeat() 复制字符串。

STR.repeat(count);
// 参数
//    count 复制次数
// 返回值
//    复制之后的字符串

let str = "poxiao";
str.repeat(3); // "poxiaopoxiaopoxiao"

补全 - padStart / padEnd#

padStart() padEnd() 是 ES2017 引入的字符串补全长度的方法。如果某个字符串不够指定长度，会在头部或尾部补全。

STR.padSrart(length, pad);
STR.padEnd(length, pad);
// 参数
//    length 指定字符串的最小长度
//    pad    用来补全的字符串
// 返回值
//    补全之后的字符串
//    如果原字符串的长度，等于或大于指定的最小长度，则返回原字符串。

let str = "poxiao";
str.padStart(2, "1"); // "poxiao"
str.padStart(10, "is"); // "isispoxiao"

替换 - replace#

如果没有使用正则表达式，只能替换一次。

这是用于搜索和替换的通用方法，是最有用的方法之一。它是搜索和替换字符串的瑞士军刀。

当 replace 的第一个参数是字符串时，它仅替换第一个匹配项。

删首尾空格 - trim / trimStart / trimEnd#

trim() trimStart() trimEnd() 去除字符串的头尾空格。

STR.trim();
STR.trimStart();
STR.trimEnd();
// 无参数
// 返回处理之后的字符串（不会改变原字符串）

let str = "   poxiao is pig   ";
str.trim(); // "poxiao is pig"
str.trimStart(); // "poxiao is pig   "
str.trimEnd(); // "   poxiao is pig"

转换 - valueOf / toString#

valueOf() 返回对象的字符串、数值或者布尔值表示。

toString() 返回对象的字符串表示。

ES Math#

取整#

// parseInt 保留整数部分
parseInt(3 / 2); // 1
// floor 向下取整（丢弃小数部分）
Math.floor(3 / 2); // 1
// ceil 向上取整
Math.ceil(3 / 2); // 2
// round 四舍五入
Math.round(3 / 2); // 2

获取随机数#

获取 0~1 之间的随机数。

Math.random();

获取指定范围的随机整数（常用）

// 公式（MAX，最大值；MIN，最小值）
const Result = Math.floor(Math.random() * (MAX - MIN + 1) + MIN);
// 简写
const Result = ~~(Math.random() * (MAX - MIN + 1) + MIN);

为什么不用 ceil 和 round 取整而用 floor

  • 用 ceil 的话，第一个数被随机到的概率极小。
  • 用 round 的话，第一个和最后一个的概率比中间各项的概率低 50%。

函数#

函数是可以反复使用的并且命名的代码块。本质上也是一种属于对象类型的数据。与其他数据类型不同的是，函数的值是可执行的代码。分为系统函数和自定义函数。

函数是数据，函数名是这个数据的标识。需要访问函数的数据去掉圆括号就好。一个函数可以有多个标识。

1、函数的作用#

  • 解决代码的冗余问题，实现代码的复用。
  • 封装代码，让函数内部的代码对外部不可见。
  • 可以让整个代码项目通过函数模块化。

2、新建函数#

函数名定义的规则遵循标识符的规则。函数的声明函数名可以重复，一旦重名，前面的覆盖后面的。

2.1 声明函数 常#

function name(args) {
  // 函数体
  return statement; // 返回语句
}

// name -- 函数名
// args -- 参数列表（可选）

var 和 function 都是声明语句，它们声明的变量和函数都在 JS 预编译时被解析，也被称为变量提升和函数提升。

ES5 中，不允许函数书写在非函数的代码块中（比如 if 语句中）。

2.2 函数表达式 常#

// 赋值调用（被称为函数表达式，这种方法只能提升函数名，不能提升函数体）
// 调用需在创建之后
let fun = function () {
  // 函数体
};

函数可以不书写名字，这种被称为匿名函数。

执行方法：

// 自调用
(function () {
  // 函数体
})();

!(function () {
  // 函数体
})();

2.3 构造函数式 了解#

类如 const x = new XXXX() 的形式，被称为构造函数。

const fun = new Function("执行字符串");
// 这个定义函数的方法几乎不用（会执行两次）

2.4 箭头函数 ES6 常#

  • 内部没有this ，箭头函数与包围它的代码共享一个this。
  • 不能用于构造函数（不能被new 实例化）。
  • 内部没有没有argument对象，可用rest参数代替。
  • 箭头函数不能使用yield命令，即不能用作Generator函数。
// 1、函数有参数，函数体只有一句返回值
function fn1(n) {
  return n + 1;
}

const fn1 = (n) => n + 1;

// 2、函数没有参数，函数体只有一句返回值
function fn2() {
  return "hello world";
}
const fn2 = () => "hello world";

// 3、函数参数随意，函数体不仅仅是一句返回值
function fn3() {}
const fn3 = () => {};

// 用箭头函数和reduce数组求和
const arrSum = (arr) => arr.reduce((p, n) => p + n);

3、使用函数#

函数需要调用才会执行。

4、参数#

JS 数据传递只有值传递，没有引用传递。

4.1 形参与实参#

形参 形式上的参数，声明函数时候提供的参数。

实参 在调用时，实际使用的参数。

4.2 默认参数#

// ES6之前
function f(name, age) {
  age ? age : (age = 18);
}
// ES6
function f(name, age = 18) {}

4.3 剩余参数#

// 可以把不确定的参数表示成数组
function f(a, ...b) {
  return b;
}
f(1, 2, 3, 4); // [2,3,4]

4.4 argument 对象#

二者在传值的时候是一一对应的。二者的个数可以不相等，形参多的时候，多出来的自动被赋值 undefined，实参多的时候，多出来的放在**arguments**对象中。该对象在函数内部才有，实际上是一个类数组 array-like 的对象，可以访问这个对象获得传递给函数的每一个实参。函数中的形参可写可不写，即使不写形参也可以使用 arguments 对象获得，使用形参比较方便。arguments 对象的 length 属性用来确定传递进来多少个实参。可以通过 length 属性的值的不同（根据传入的参数个数的不同），进行不同的操作。arguments对象中保存着一个叫callee的属性，这个属性是一个指针，它指向拥有这个arguments对象的函数。

  • 使用 arguments 对象的 length 属性可以获取函数的实参个数。
  • argument 对象只能在函数内可见,因此 arguments.legth 也只能在函数体内使用。
  • 使用函数对象的 length 属性可以获取函数的形参个数,该属性为只读属性,在函数体内,体外都可以使用

callee 是 arguments 对象的属性，它引用当前 argument 对象所在的函数，使用该属性可以在函数体内调用自身。

// 检测函数的形参和实参是否一致
arguments.length != arguments.callee.length;

5、返回值#

return 没有指定值时函数返回的是undefined，函数在执行完该语句以后立即停止执行并退出函数。

返回值是一个函数的函数称为高阶函数。

ECMA-262 规范了7种错误类型。其中Error是基类，其他6种错误类型是子类，都继承了基类。基类的主要作用是自定义错误对象。

ReferenceError 读取不存在的变量时抛出
TypeError 值的类型发生错误的时候抛出
RangeError 数字超出合法范围

2、try…catch#

try{
    // 调试代码块
} catch (e) {
    // 捕获异常，并进行异常处理代码块

四、逻辑运算符#

&& 与

  • 第一步：计算第一个操作数
  • 第二步：如果第一个操作数转换成 false，那么就会结束运算，直接第一个操作数
  • 第三步：如果第一个操作数返回 true，则计算第二个操作数的值
  • 第四步：第二个操作数如果返回 true 则逻辑与返回第二个操作数，否则返回第一个操作数

|| 或

  • 第一步：计算第一个操作数的值。
  • 第二步：检测第一个操作数的值。如果左侧的表达式的值可以转换为 true，那么就会结束运算。直接返回第一个操作数的值
  • 第三步：如果第一个操作数可以转换为 false，则计算第二个操作数的值。并返回第二个操作数的值

! 非

1 && 2; // 返回2

"" || false; // f || f 返回 false
false || ""; // f || f 返回 ""

在逻辑运算中，如果第一个操作数就能决定结果，那么第二个操作数就不会运行了。（就算运行了会报错也能顺利通过）

一、使用#

1、不用加 new

```js

4、Symbol.for / Symbol.keyFor
有时，我们希望重新使用同一个Symbol值，Symbol.for()方法可以做到这一点。它接受一个字符串作为参数，然后搜索有没有以该参数作为名称的 Symbol 值。如果有，就返回这个 Symbol 值，否则就新建一个以该字符串为名称的 Symbol 值，并将其注册到全局。

let s = Symbol.for('foo')
let s2 = Symbol.for('foo')
s1 === s2 // true

Symbol.keyFor () 方法返回一个已登记的 Symbol 类型值的 key。

let s = Symbol.for('foo')
Symbol.keyFor(s) // "foo"

let s2 = Symbol('foo')
Symbol.keyFor(s2) // undefined

ES 开始#

JavaScript 最初的目的是为了赋予网页生命。
这种编程语言我们称之为脚本。它们可以写在 HTML 中，在页面加载的时候会自动执行。脚本作为纯文本存在和执行。它们不需要特殊的准备或编译即可运行。

  • 1995 JavaScript、JScript 诞生
  • 1997 ECMA 给 JS 制定了标准ECMA-262，命名为ECMAScript，即 ES1.0。ES 是 JS 的规范标准，JS 是 ES 的实现。
  • 1999 发布 ES3，成为 JS 通用标准。
  • 2009 发布 ES5。
  • 2013 ES6 草案发布（其实就是之前不被接受的 ES4 激进方案）
  • 2015 发布ES6，即 ES2015，从此 JS 新版本按 ES+年份形式发布。ES6 是一个历史名词，也是一个泛指，代表 ES6 以后的新一代标准。

一、特点#

弱类型 数据类型不固定，可以随时改变。
解释型 解释型是一行一行解释运行，编译型是需要整体编译。
脚本语言 脚本是需要载体运行的。
基于对象/原型 只有对象，没有类，对象继承对象，而不是类继承类。原型对象是基于原型语言的核心概念。原型对象是新对象的模板，它将自身的属性共享给新对象。一个对象不但可以享有自己创建时和运行时定义的属性，而且可以享有原型对象的属性。

二、组成#

JavaScript 由 ECMAScript、BOM、DOM三部分组成，其中 ECMAScript （简称ES）是JS的核心语法规范。

  • ECMAScript 核心语法，简称 ES。
  • DOM Document Object Model 文档对象模型。
  • BOM Browser Object Model 浏览器对象模型。

三、网页中使用 JS 的方法#

（1）页面嵌入（常用）

<script>
// 这里写JS代码
</script>

（2）行内写法（几乎不用，局限性大，代码分离性差，可读性不强）
（3）外部文件引入（常用）

script 标签属性

  • src 指定外部脚本的 URI，如果设置了，标签内容会忽
  • type 指定脚本的语言，默认为 JS。
  • async 规定一旦脚本可用，则会异步执行。
  • defer 是否对脚本执行进行延迟。
<script src="链接"></script>

  • 控制台日志: console.log()、console.dir()
  • 系统对话框（弹窗）: alert()、confirm()、prompt()
  • 直接显示在文档上: document.write
