181003：Markdown 使用心得

发表于 2021-01-28 更新于 2021-02-01 分类于 1810：Markdown

Markdown 使用心得#

1、链接的文件名字带空格不识别怎么办？

解决: 在链接中用%20替代空格的书写

2、永久插入图片

首先将图片通过在线转码工具转换成base64的编码，并用以下格式嵌入即可，格式如下：

![image] (base64)

但是由于这base64编码往往都很长，很占篇幅，因此我们可以给图片编号，并将所有的图片base64编码放在文档的最后即可，格式如下：

  • 在插入图片的地方使用：![image] [图片编号]
  • 在文档最后使用：[图片编号]:base64编码

这样即可完成图片的永久插入！

181004：Typora

发表于 2021-01-28 更新于 2021-02-01 分类于 1810：Markdown

Typora#

  • 源代码模式：CTRL + /
  • 打字机模式：F8
  • 专注模式：F9
  • 显示/隐藏侧边栏：CTRL + SHIFT + L
  • 段落：CTRL + 0
  • 标题：CTRL + 1/6
  • 表格：CTRL + T
  • 选中一整行：CTRL + L
  • 选中单词：CTRL + D
  • 跳转到文章开头：CTRL + HOME
  • 跳转到文章结尾：CTRL + END
  • 插入链接：CTRL + K
  • 设置：CTRL + ，
  • 返回顶部: CTRL + Home

1、专注模式#

快捷键 F8 打开专注模式，编辑器会高亮显示当前正在编辑的段落，让你更加专注于当前内容，不受视觉干扰。

2、打字机模式#

快捷键 F9 打开打字机模式，当前正在编辑的段落会居中显示。

3、数学公式#

数学的公式块是使用 LaTeX 语言实现，输入$$，然后按下Enter键就会弹出一个支持 TeX/LaTeX 语法的输入框。

Typora Mermaid#

Mermaid 是一种简单的类似 Markdown 的脚本语言，通过 JavaScript 编程语言，将文本转换为图片。因此，真正实现画图功能的并不是 Typora 本身，它只是内置了对 Mermaid 的支持。

时序图#

语法解释：->> 代表实线箭头，-->> 则代表虚线。

sequenceDiagram

    John-->>Alice: Great!

Hexo#

https://hexo.io/

一、开始#

1、安装#

npm install hexo-cli -g

2、新建博客#

hexo init 博客名称
cd 博客名称
npm i

3、常用命令#

# 启动博客（本地服务器）
# 打开 localhost:4000 查看
hexo s

# 清除缓存
hexo clean

# 构建
hexo g

4、博客配置#

博客根目录下的 _config 为 博客配置文件

主题目录下的 _config 为 主题配置文件

5、主题配置#

主题库

这边推荐 NEXT 主题。

安装NEXT主题#

git clone https://github.com/theme-next/hexo-theme-next themes/next
cd themes/next
npm i

修改 博客根目录下 _config 文件 里面的主题为 next

6、Github Page / Gitee Page#

npm install hexo-deployer-git --save
# _config.yml

# git 配置
deploy:
  type: git
  repo: https// .... .git
  branch: master

推上去

hexo d
