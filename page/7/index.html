CSS 选择器#

要使用 CSS 对 HTML 页面元素实现一对一，一对多，多对一的控制，就要用到选择器。

一、基本选择器#

1. 元素选择器#

也叫标签选择器。

通过元素标签名字选择页面中所有这个标签的元素。

2. 类选择器#

class 属性用来给元素分组，是允许多个元素拥有的，且一个元素可以拥有多个class。

通过 . + 元素的class属性进行选择。

<style>
  .qwq {
    ...
  }
</style>
...
<div class="qwq"></div>
<!-- 允许书写多个类名，用空格分隔 -->
<div class="qwq1 qwq2"></div>

3. id 选择器#

id相对class来说，是不允许重复的，且一个元素只能有一个id属性。

id 属性命名规范：

  • 以字母开头
  • 不出现汉字、空格
  • 不推荐带有ad 相关的英文单词，容易被浏览器广告过滤插件屏蔽
  • 见名知意

通过 # + 元素的id属性进行选择。

<style>
  #qwq {
    ...
  }
</style>
...
<div id="qwq"></div>

4. 通配符选择器#

表示选择所有的元素。

几乎不用，影响性能。

* {
  ...
}

二、关系选择器#

1. 群组选择器（并集选择器）#

用 ,就能对多个选择器一起设置样式。

2. 交集选择器#

由两个选择器直接连接构成，其结果是选中二者各自元素范围的交集。其中第一必须是标签选择器，第二个必须是类选择器或者id选择器，两个选择器之间不能有空格，必须连续书写。

如：p#xxx 或者 p.xxx。

3. 子级选择器#

用>隔开。如 A>B，在A元素中寻找子元素是B的。

4. 后代选择器#

用空格隔开。如 A B，意思是在A元素中寻找后代是B的元素。

5. 兄弟选择器#

相邻兄弟选择器 A+B，意思是在A元素下紧贴这A的并且是B才能被选中。

通用兄弟选择器 A~B，在A元素下边的兄弟元素，并且是B就能被选中。

三、属性选择器#

选择器 描述
E[attr] 有属性attr的所有E元素
E[attr='val'] 属性attr的值是val的元素
E[attr^='val'] 属性attr的值以val开头的元素
E[attr$='val'] 属性attr的值以val结尾的元素
E[attr*='val'] 属性attr的值包含val字符串的元素

四、伪类选择器#

1. 动态伪类选择器#

  • 锚点伪类：link、visited
  • 用户行为伪类：hover、active、focus
  • 目标伪类：target（IE9及以上）

2. 状态伪类选择器#

选择器 描述
E:enabled 匹配表单中可用的元素
E:disabled 匹配表单中禁用的元素
E:checked 匹配表单中被选中的radio（单选框）或checkbox（复选框）元素

3. 结构性伪类选择器#

选择器 描述
E:first-child 选择第一个子元素E
E:last-child 选择最后一个子元素E
E:nth-child(n) 选择一个或多个特定的子元素 （第一个编号为1）
E:nth-last-child(n) 选择一个或多个特定的子元素，从最后一个子元素开始算
E:only-child 匹配父元素下仅有的一个子元素
E:empty 匹配一个不包含任何子元素的元素 ( 文本节点也被看作子元素 )
E:not(s) 匹配不符合当前选择器的任何元素( 反选 )

五、伪元素选择器#

:before

:after

待更新。

六、选择器的优先级与权重#

1. !important#

提高指定样式规则的优先级为最高优先级。

写在定义的最后面。如：

.xxx {
  color: #fff !important;
}

2. 优先级顺序#

从上到下，依次增加。

  • 默认 / 继承（inherit）的样式
  • 通配符
  • 标签选择器
  • 类选择器
  • id 选择器
  • 行内书写的样式
  • !important

当优先级相同时，后写的样式优先级更高。

3. 权重计算#

类型 优先级权重
!important ∞
行内书写 1000
id 选择器 100
类选择器、伪类选择器、属性选择器 10
标签选择器、伪元素选择器 1

CSS 开始#

CSS，即 Cascading Style Sheets，层叠样式表。

具有：

  • 像素级精确控制。
  • 将内容和表现形式分离。
  • 层叠特性。

一、基本语句#

通常分为三部分：选择器，属性，属性值。

       属性      分号
.box { width:10px;  height:20px; }  
选择器        属性值  可以写多个

二、使用方式#

1. 内联式#

一般不推荐使用。

<div style="width:10px; height:10px;"></div>

特点：

  • 优先级高
  • 臃肿，代码冗余（不能复用）
  • 结构和样式没有分离，不方便维护

2. 头部引入#

常用。

<head>
  ...
  <style>
    div {
      width: 10px;
      height: 10px;
    }
  </style>
</head>

特点：

  • 方便维护和修改
  • 样式和结构分离

3. 外部引入#

（1）link 方式

<head>
  ...
  <link rel="stylesheet" type="text/css" href="index.css" />
</head>

（2）@import 方式

不推荐。

（3）@import 和 link 方式引入的区别：

  • link 功能较多，比如可以定义 RSS，而 @import 只能用于加载 CSS
  • 当解析到 link 时，页面会同步加载所引用的 CSS，而 @import 所引用的 CSS 会等页面加载完才被加载
  • link 可以用 JS 动态加载 CSS，而 @import 不允许
  • @import 在 IE5 以上才能使用

1、form#

<form action="" method="POST"></form>

action：表单提交接口。如果省略，则设置为当前页面。

method：表单提交的方式。一般为 POST 或者 GET 。

  • selected 预先选择一个选项。
  • multiple 实现多选。

HTML 标签#

一、概述#

标签，又称为元素。

1、标签的关系#

兄弟关系
父子关系

2、标签的分类#

（1）块标签 block

  • 代表标签: div
  • 默认的宽度是占据父标签全部内容，即 width: 100%
  • 默认的高度由内容决定
  • 设宽高、边距、边框没有限制
  • 独占一行
  • 允许嵌套块元素和行元素（p/h标签除外）

（2）行内标签 inline

  • 代表标签: span
  • 默认的宽高由内容决定。
  • 内容过多时允许换行并断开（一般都是用于处理文字样式）。
  • 设置宽高无效。上下边距、边框无法设置。
  • 共用一行：在一行中如有剩余空间，允许同其他行内/行内块标签共用。
  • 某些动画样式不能添加和使用。

（3）行内块标签 inline-block

  • 代表标签: img、button、input
  • 默认的宽高由内容决定。
  • 内容过多时不会断开。
  • 可以设置宽高。
  • 在一行中如有剩余空间，允许同其他 行内/行内块 标签公用一行。
  • 行内块和内行标签会有行高的概念。
  • 行内块受父元素的 line-height 以及自身的 vertical-align 影响。

将非块级元素转换成块级元素的方法

  • display: block
  • float: left/right
  • position: absolute/fixed
  • 父级元素使用 display:flex

3、语义化标签#

优点：

  • 语义化标签更具有可读性，便于团队的开发和维护
  • 没有 CSS 的情况下，网页也能很好的呈现出内容结构和代码结构
  • 关于SEO，搜索引擎更能理解到网页中各部分之间的关系，更准确更快速搜索信息

布局的三个发展阶段：

  • 开荒时期：前端的代码主要是后台写的，使用 table 来布局
  • 美工时期：有专门的人来写前端代码了，主要布局方式是 DIV+CSS
  • 现在：使用具有语义的标签，可以表达正确的内容，也利用开发和维护。

HTML & 网页基础#

一、基本结构#

<!-- 版本声明 -->
<!DOCTYPE html>
<!-- 根文档 -->
<html lang="en">
  <!-- 头部 -->
  <head>
    <meta charset="utf-8" />
    <title>标题</title>
  </head>
  <!-- 主体 -->
  <body></body>
</html>

1、注释#

用<!-- 和-->包裹起来。

2、文档版本声明#

<!DOCTYPE html>

给浏览器声明，告诉浏览器应该按照 HTML5 的规范解析当前页面。 如果不写，浏览器就是按照自己默认的模式解析。

版本声明必须写在第一行。我们不把他看做一个 html 标签，而是把它当做一个指令，并且这个指令对大小写不敏感。

<!-- html 4 的 版本说明(了解) -->
<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

3、html 、head、body 标签#

html 是一个双标签，所有网页内容全部书写在 html 标签中。lang就是 html 标签的属性，en就是lang属性的值，属性是对当前标签的信息的扩展。

head 标签代表网页的头部，不会显示在网页中。主要是包含了标题，网页的描述，字符编码等等一些信息。

body 标签是网页显示的主体内容。

4、meta 标签#

控制网页的元信息（信息的信息），根据属性的不同，有很多的作用。

5、解决网页乱码问题#

<meta charset="utf-8" />

charset即字符编码。utf-8称作为万国码。
