CSS 浮动

发表于 2020-06-16 更新于 2020-08-26 分类于 技术栈

一、浮动#

float 浮动 可以让多个块元素横向排列，共行。

属性 说明
none 默认，不浮动
left 左浮动
right 右浮动

1、概念#

正常页面流/正常文档流 是行元素从左至右，块元素从上到下布局。浮动 是破坏流（脱离文档流）的一种方式。在目前开发中，很多的元素都是一个个float堆叠起来，但是这种布局很容易崩溃。只要宽度和高度发生一点变化，页面就会发生错乱。

2、浮动的使用#

左浮动 就是让容器沿着容器左侧排序。

右浮动 让容器沿着容器右侧排序。

3、高度塌陷#

浮动元素以后，脱离父级内容区域，父级没有内容撑开自身的高度，而且父级的兄弟元素是按照父级的位置进行布局的，所以页面会乱。

4、清除浮动#

清除浮动不是不让元素浮动，是清除浮动对父级高度塌陷的影响。

清除浮动的方法

  • 给浮动元素的父级设置高度。
  • 给浮动元素的父级设置浮动，以浮制浮。
  • overflow设置不为visible的属性。
  • 在最后一个浮动元素的后边添加一个空的块级元素，添加一个块级元素，添加属性和值为 clear: both。
  • （推荐）after伪元素清浮动，给所有浮动元素父级一个clearFix的类名。
