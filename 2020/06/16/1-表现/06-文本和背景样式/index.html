CSS 样式设置之文本和背景

发表于 2020-06-16 更新于 2020-08-27 分类于 技术栈基

一、文本和字体样式#

1、color 设置文本颜色#

（1）可设置颜色的类型

类型 示例 说明
英文名 red 开发中很少出现
RGB RGBA rgb(255,255,255)
十六进制 #112233 常用
HSL HSLA hsl(0,0,0) H色调：0-360
S饱和度 0-100%
L亮度 0-100%
inherit 继承（默认值）
transparent 透明

（2）a标签的color属性默认不是继承，需要的时候需直接控制。

2、font-family 字体控制#

3、@font-face 自定义字体#

4、font-size 字号大小#

常用单位

  • px
  • em 参考父级的字号大小 前边的数字是倍数
  • rem 参考根元素的字号大小（谷歌浏览器默认字号大小16px）

注：谷歌浏览器默认16px，默认最小字号支持是12px。

5、font-style 字体风格#

  • normal 正常
  • italic 斜体
  • oblique 倾斜（火狐不支持）

6、font-weight#

控制文字粗细。（可继承）

值 描述
normal 正常
bold 加粗
lighter 相对父级细一级
bolder 相对父级粗一级
100~300 细
400~500 正常
600~900 粗

7、font 合写#

font-style -weight -size/line-height -family
字号大小和字体必写，隐藏一个line-height属性，所以不继承的时候，需要检查font合写。

8、line-height 行高控制 ==重要==#

其实是控制两行文字基线的距离，==决定了内联元素的占用的高度==。默认会继承。

  • 固定值 px 不能是负
  • 比例：比如1.5 就是当前文字大小的1.5倍
  • 百分比：80%
  • 浏览器默认值：normal

让单行文本垂直居中：行高 = 高度。

9、vertical-align 垂直对齐方式#

控制某个行内元素在垂直方向的对齐方式。

baseline 基线（默认）
middle 中线
top 顶线
bottom 底线

图片垂直居中

10、text-align 文本对齐设置#

text-align 文本对齐设置。控制当前元素内容。

属性值

  • left 默认值，左对齐
  • right 右对齐
  • center 居中对齐
  • justify 两端对齐

控制范围 ==无法让块元素居中==

  • 元素内的文本
  • 行内块元素
  • 行元素

11、text-decoration 文本修饰设置#

none 去掉下划线
underline 下划线
line-through 删除线
overline 上划线

线条的颜色是默认跟随文字颜色改变的。text-decoration 是间写属性，后边可以跟三个值：线的位置 线的颜色 线的类型。

12、text-indent 文本缩进设置#

text-indent 文本缩进设置

属性值

  • 推荐单位为em， px不方便控制。
  • 可以为负，实现悬挂缩进的效果。

控制范围

  • 只针对块元素生效

13、letter-spacing、word-spacing：#

letter-spacing
控制字符间距
控制所有字符和字符之间的间距

word-spacing
控制词间距
改变词和词之间的间距
以空格来分割两个词 值都可以为负

14. white-space 空白处理#

值 描述
normal 默认。空白会被浏览器忽略。
pre 空白会被浏览器保留。
其行为方式类似 HTML 中的 <pre> 标签。
nowrap 文本不会换行，文本会在在同一行上继续，直到遇到 <br> 标签为止。
pre-wrap 保留空白符序列，但是正常地进行换行。
pre-line 合并空白符序列，但是保留换行符。
inherit 继承

二、背景样式#

任何背景是撑不开元素的宽高。

1、background-color 背景颜色#

背景颜色是从边框下开始显示的。

设置值同 color。

2、backgroud-image 背景图片#

background-image: url(路径)
边框下也显示图片，但是是平铺出来的。
背景图片的左上角原点，是以padding区域的左上角开始显示的。

3、background-position 背景图定位#

由两个值构成（background-position-x background-position-y），值可以是 （坐标轴是以padding区域的左上角原点的坐标）

  1. px
  2. 关键字
    在x轴上有 left right center
    在y轴上有 top bottom center
  3. 百分比
    当百分百的时候，和right bottom效果一致

4、background-repeat 背景图平铺#

repeat 默认 x和y都平铺
repeat-x 只沿着x轴平铺
repeat-y 只沿着y轴平铺
no-repeat 不平铺

5、背景合写#

background: color image position-x position-y repeat
如果有background合写属性，那么当此元素还有关于background属性的时候，请书写在合写的下边。

6、背景图原点#

background-origin: padding-box (默认)

  • border-box | padding-box | content-box
  • background-origin是用来决定图片的原始起始位置。
  • 它有三个可选值content-box,padding- box,border-box，即可以选择背景图片是从内容区域或者内边距或者边框开始显示。

7、背景图裁剪#

background-clip

  • border-box | padding-box | content-box
  • background-clip的作用为将背景图片做适当的裁剪，以适应需要。
  • background-clip有content- box,padding-box,border-box，text几个值剪裁方法：根据设置的盒子部位，那么图片在这个部位的外边缘以外的部分都会不可见。

8、背景图大小#

background-size

  • length： 长度值—第一个值设置宽度，第二个值设置高度
  • percentage： 百分比—第一个值设置宽度，第二个值设置高度
  • cover：充满容器，比例不发生变化，但是有可能超出。
  • contain： 将背景图像等比缩放到宽度或高度与容器的宽度或高度相等，背景图像始终被包含在容器内

9、多重背景 C3#

CSS3允许为元素使用多个背景图片。

元素引入多个背景图片，前面图片会覆盖后面的图片。
