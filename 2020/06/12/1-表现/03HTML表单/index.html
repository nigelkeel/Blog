一、基本表单结构#

1、form#

action：表单提交接口。如果省略，则设置为当前页面。

method：表单提交的方式。一般为 POST 或者 GET 。

  • GET 通过URL提交
  • POST 通过请求报文

2、input 标签#

0 属性#

属性 描述
placeholder 占位符。控制元素为空时的提示信息，只要输入内容就会消失。
  • name 当前表单元素键名。
  • value 当前表单元素输入内容，默认内容。
  • type 表单元素的类型。
  • autocomplete 是否提示用户输入历史输入值，默认为 ON。
  • autofocus 自动获取焦点。
  • required 必填。提交表单时必须填写此项。
  • disabled 禁用表单元素。不会被提交。
  • readonly 只读。会被提交。
  • checked 预先选中单选框或者多选框。
  • form 关联表单。如果input存在form属性，表示该input属于某个表单。通过input的form属性为form表单的id，进行关联。 进行关联以后，无论input书写在哪里，都能随着表单提交数据。
  • selected 预先选择一个选项。
  • multiple 实现多选。

3、button 按钮#

有3种 type，分别是 button，submit，reset。

具体功能需要配合JS实现。

注1：input提交按钮和button提交按钮的区别：input是单标签，button是双标签。
注2：button是行内块标签。

4、label 标签#

定义表单元素标注，获取焦点。

两种方式来绑定表单。

5、select-option 下列列表#

在 option 中 value 属性书写的内容不会显示，但是最后会提交。显示的内容写在 option 标签中。

selected 默认选中

multiple 多选

二、表单补充#

1、H5 表单新规范#

  • 表单的属性名和属性值相等的时候，可以省略属性值，直接书写属性名即可。
  • 在JS中，允许用布尔值[^6]来开启和关闭属性。
  • 在多个表单元素，只能出现一次的属性，如果书写多个，以第一个为准。
