一、JSON

JSON 是ES内置对象（不用定义就能用）。

json 是一种常用于前后端数据交互的格式，本质是字符串。

1、json 值

可以表示三种类型数据：

（1）简单值

除了 undefined 之外的基本数据类型，表示字符串必须使用双引号。

（2）对象 表示的是一组无序的键值对，属性必须用双引号包起来。

（3）数组 有序的集合、键值对。

2、json 数据处理 ==常用==

用于前后台发送数据。

二、Math ==需补充==

Math 是JS内置的数学处理对象。

方法 描述
Math.ceil() 进一取整
Math.floor() 舍一取整
Math.round() 标准的四舍五入
Math.max() 获得参数列表的最大值
Math.min() 获得参数列表的最小值
Math.abs() 绝对值
Math.pow(底数，次方) 幂运算(**^21)
Math.PI 圆周率π
Math.random() 返回大于等于0小于1的一个随机数
Math.sign() 判断正负
正数 1
零或者null 0
负数 -1
其他 NaN

平分根、立方根 Math.sqrt() Math.cbrt() Math.hypot(3, 4)==5

  • Math.ceil() 进一取整

三、Date ==需补充==

Date 是JS中用来处理日期和时间的内置对象。

1、时间戳

时间戳是一个整数，保存1970年1月1日到现在的毫秒数。

自动校准机制：超过范围的会自动换算。

2、时间处理方法

方法 功能
getFullYear 获得四位年份
getMonth 获取月份，==0-11==
getDate 获取月份中的天数，1-31
getDay 获取星期，==0表示周日，0-6==
getHours 小时，0-23
getMinutes 分钟，0-59
getSecond 秒数，0-59
