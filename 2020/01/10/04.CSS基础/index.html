一、引用方式

CSS 即层叠样式表。它提供像素级精确样式控制，将内容和表现形式分离， 以及 层叠 特性。

1. 内联引用

  • 臃肿，冗余代码多。
  • 结构和样式没有分离，不方便维护。
  • 优先级高。

2. 头部引用

  • 方便维护和修改
  • 样式和结构分离
  • 服务器压力小，大型门户网站首选
  • 相对整站有冗余代码，相对单页没有

三、关系选择器

1、后代选择器、子代选择器

后代选择器使用空格，比如A B。

子代选择器使用>，如A>B。

2、兄弟选择器

（1）相邻兄弟选择器

A+B

使用+号间隔开两个基础选择器，A+B选择紧邻A选择的下一个兄弟元素没并且是B元素。

（2）通用兄弟选择器

A~B

使用~号间隔开两个基础选择器，A~B选择A后边所有是B的兄弟元素。

五、属性选择器

选择器 描述
E[attr] 有属性att的所有E元素
E[attr='val'] 属性att的值是val的E元素

六、结构性选择器

选择器 描述
E:first-child 选择父级中第一个子元素
E:last-child 选择最后一个子元素
E:nth-child(n) 选择一个或多个特定的子元素(第一个编号为1)
E:nth-last-child(n) 选择一个或多个特定的子元素，从最后一个子元素开始算
E:nth-of-type(n) 选择同类型的某个特定元素
E:only-child 匹配父元素下仅有的一个子元素
E:empty 匹配一个不包含任何子元素的元素(文本节点也被看作子元素)
E:not(s) 匹配不符合当前选择器的任何元素(反选)

注意对象是E元素的父级。
假如div中第一个元素不是p，第二个元素是p，那么div p:nth-child(1)无法选中。此时需要使用nth-of-type

八、样式重置

样式重置（CSS reset），将html的默认样式全部去掉，需要的时候我们自行添加。统一页面风格。

为什么要用样式重置

  • 多数元素拥有自己的默认样式，并且我们很多都不需要。
  • 默认样式在不同浏览器中呈现的也不一定相同，就会导致浏览器展示页面不相同的现象。
  • 整个页面中固定的风格代码，可以在样式重置中直接书写，不用每一个都设置。

为什么不用通配符重置

  • 重置简单，不够完整，一些demo可以使用，真正项目不推荐使用，请使用CSS reset
  • 通配符，匹配所有的标签，影响很大，效率很低。

九、文档流

脱离文档流，指的是元素脱离正常元素的布局排版规则，其他处于文档流中的盒子在计算布局排版时，会自动无视已脱离文档流的元素来进行定位。

浮动与绝对定位之间的区别就是：

  • 浮动会使元素脱离文档流，但是不会脱离文本流，在于其他盒子的文本内容计算布局的时候，还是占位置的
  • 绝对定位会使元素脱离文档流，同时也会脱离文本流，在于其他盒子的文本内容计算布局的时候，不占位置。
