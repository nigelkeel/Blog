正则表达式#

正则表达式（RegExp，Regular Expression），又叫规则表达式，是由一些特殊符号（元字符、限定符）组成的式子。

是搜索和替换字符串一种强大的方法。在 JS 中，正则表达式通过内置的 RegExp 类的对象来实现，并和字符串集成。用来匹配字符串。

一、开始#

1、创建一个正则表达式对象#

// （1）允许从字符串动态构造模式
reg = new RegExp("pattern", "flags");

// （2）常用
reg2 = /pattern/;
reg3 = /pattern/gim;

如果要在字符串中搜索，可以使用 search() 。这个方法返回匹配项的索引位置，如果没找到返回-1。

// 例子

let str = "I love you!";
let reg = /love/;

console.log(str.search(reg)); // 2

test()

// 匹配就是 true
// 不匹配就是 false

2、元字符#

元字符 表述
. 除了\n之外任意单个字符
[] 表示一个范围 / 去除元字符串原有的作用
- [0-9] 0-9 之间任意的单个数字
- [a-zA-Z] 表示任意一个字母
- [.] 表示单纯的.
| 或者
- [a-z] | [0-9] 小写字母或者数字
() 提升优先级 / 分组（第几组从左开始数(）
\b 单词的边界
\d / \D 数字 / 非数字
\s / \S 空白字符 | 非空白符
\w / \W 非特殊符号 | 特殊符号（下划线不是特殊符号）
^ 开头 / 取反（在[] 里面）
$ 结尾

3、限定符#

限定符也是一种元字符，限定某些表达式出现的次数。

限定符 表述
{} 该限定符前面的表达式出现的次数
- [a-z]{5} 出现 5 次
- [a-z]{5,} 出现 5 次或者多次
- [0-9]{0,1} 出现 0 次或者 1 次
* 表示前面的表达式出现 0 次或者多次
+ 表示前面的表达式出现 1 次或者多次
? 表示前面的表达式出现 0 次或者 1 次

4、修饰符#

正则表达式的修饰符影响搜索结果，在 JS 中有五个修饰符：

修饰符 描述
i 搜索不区分大小写
g 搜索会查找所有的匹配项，而不是第一个
m 多行模式
u 开启完整的 unicode 支持
y 粘滞模式 
let str = "I love you!";

console.log(str.search(/LOVE/)); // -1
console.log(str.search(/LOVE/i)); // 2

搜索默认是区分大小写的。
