ES Math#

取整#

// parseInt 保留整数部分
parseInt(3 / 2); // 1
// floor 向下取整（丢弃小数部分）
Math.floor(3 / 2); // 1
// ceil 向上取整
Math.ceil(3 / 2); // 2
// round 四舍五入
Math.round(3 / 2); // 2

获取随机数#

获取 0~1 之间的随机数。

Math.random();

获取指定范围的随机整数（常用）

// 公式（MAX，最大值；MIN，最小值）
const Result = Math.floor(Math.random() * (MAX - MIN + 1) + MIN);
// 简写
const Result = ~~(Math.random() * (MAX - MIN + 1) + MIN);

为什么不用 ceil 和 round 取整而用 floor

  • 用 ceil 的话，第一个数被随机到的概率极小。
  • 用 round 的话，第一个和最后一个的概率比中间各项的概率低 50%。

3、最值#

Math.max() Math.min() 获取一个参数列表的最大值，最小值。

// 获取一个数组的最值
let arr = [1,2,3,4,5,6,7]

// ES5
Math.max.apply(null,arr);

// ES6
Math.max(...arr);

7、生成唯一的随机字符串#

function uniqueId() {
  // substring去掉.
  const randomNum = Number(Math.random().toString().substring(2));
  // .toString(32) 转换为32进制
  return (randomNum + Date.now()).toString(32);
}
