HTML 基础

发表于 2020-06-11 分类于 技术栈基

一、网页入门#

1、概念#

  • WWW，World Wide Web，万维网
  • B/S结构，浏览器-服务器结构
  • C/S结构，客户端-服务器结构
  • HTTP协议，超文本传输协议
  • Domain Name，域名，DNS；Domain Name System，域名解析系统
  • W3C，万维网联盟
  • HTML，Hypertext Markup Language，超文本标记语言。
  • CSS，层叠样式表。

2、开发环境#

  • VSCode
  • Chrome、Firefox、IE
  • Photoshop

3、常见浏览器及其内核#

浏览器 内核 浏览器前缀
IE Tredent -ms-
Safari Webkit -webkit-
Firefox Gecko -moz-
Chrome Blink -webkit-
Edge、Opera Blink -webkit-

注1 Edge之前采用 EdgeHTML 内核，Opera之前使用 Presto 内核。Blink内核是Webkit内核的分支。
注2 Android和iOS移动设备只能用自带系统浏览器内核。
注3 PC端浏览器市场份额Chrome浏览器排名第一，IE第二，二者约占70%。

4、网页的组成#

网页由三部分组成：

  • 结构 html
  • 表现 CSS
  • 行为 JavaScript

二、HTML 基本结构#

1、注释#

用<!-- 和-->包裹起来的内容。

2、版本声明#

给浏览器声明，告诉浏览器应该按照HTML5的规范解析当前页面。 如果不写，浏览器就是按照自己默认的模式解析。

版本声明必须写在第一行。我们不把他看做一个html标签，而是把它当做一个指令，并且这个指令对大小写不敏感。

3、html、head、body 标签#

html 是一个双标签，所有网页内容全部书写在html标签中。lang就是html标签的属性，en就是lang属性的值，属性是对当前标签的信息的扩展。

head 标签代表网页的头部，不会显示在网页中。主要是包含了标题，网页的描述，字符编码等等一些信息。

body 标签是网页显示的主体内容。

4、meta 标签#

控制网页的元信息（信息的信息），根据属性的不同，有很多的作用。

5、解决网页乱码问题 △#

charset即字符编码。utf-8称作为万国码。
